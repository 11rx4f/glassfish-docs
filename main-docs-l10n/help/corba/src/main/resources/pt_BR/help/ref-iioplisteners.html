Listeners IIOP

Utilize a página Listeners IIOP para configurar listeners IIOP.

Vários listeners IIOP podem ser configurados para um ORB, cada um deles aceitando conexões em diferentes combinações de endereço e porta de rede. Listeners podem ser configuradas de maneiras diferentes para comunicações seguras e autenticação de clientes.

Como default, três listeners de IIOP são configurados: orb-listener-1,SSL e SSL_MUTUALAUTH.

Para cada listener IIOP, a página Listeners IIOP contém as informações a seguir.

Nome

O nome do listener IIOP.

Ativado

Verdadeiro se o listener IIOP estiver ativo ou falso se o listener não estiver ativo.

Endereço de Rede

O endereço do Protocolo Internet (IP) ou o nome do resolúvel DNS para o listener IIOP.

Porta de Listener

O número da porta para o listener IIOP.

Connector

Auf der Seite "Connector" können Sie die folgenden Arten von Connector-Ressourcen konfigurieren:

  • Connector-Ressourcen

  • Connector Connection-Pools

  • Admin-Objektressourcen

  • Arbeitssicherheitszuordnungen

Ein Connector-Modul (auch als Ressourcenadapter bezeichnet) ist eine Java-Komponente, die es Anwendungen ermöglicht, mit EIS-Software (Enterprise Information System) zu interagieren. EIS-Software umfasst verschiedene Arten von Systemen: Enterprise Resource Planning (ERP), Mainframe-Transaktionsverarbeitung, nicht relationale Datenbanken und Messaging-Systeme wie z.B. Java Message Service (JMS).

Ein Connector Connection-Pool besteht aus einer Gruppe wiederverwendbarer Verbindungen für ein spezielles EIS.

Eine Connector-Ressource ist ein Programmobjekt, das eine Anwendung mit einer Verbindung zu einem EIS versorgt.

Eine verwaltete Objektressource stellt spezielle Funktionen für eine Anwendung bereit.

Eine Work Security Map ordnet einen mit einer eingehenden Arbeitsinstanz verknüpften Principal einem Principal in der Sicherheitsdomain von GlassFish Server zu.

Ressourcenadapterkonfigurationen

Auf der Seite "Ressourcenadapterkonfigurationen" können Sie Konfigurationen von Connectoren (Ressourcenadaptern) verwalten.

Die Ressourcenadapterkonfiguration liefert die Konfigurationsinformationen für den Ressourcenadapter. Mit der Administrationskonsole erstellen Sie eine Ressourcenadapterkonfiguration, nachdem der Ressourcenadapter bereitgestellt wurde. In diesem Fall wird der Ressourcenadapter mit der neuen Konfiguration neu gestartet. Um eine Ressourcenadapterkonfiguration vor der Bereitstellung des Ressourcenadapters zu erstellen, verwenden Sie den Befehl asadmin create-resource-adapter-config .

Bevor Sie eine Ressourcenadapterkonfiguration erstellen können, müssen Sie einen Threadpool erstellen, den der Ressourcenadapter verwenden soll.

Für jede Ressourcenadapterkonfiguration werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des Ressourcenadapters.

Die Tabelle "Konfigurationen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Ressourcenadapterkonfiguration.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Ressourcenadapterkonfiguration.

Verwandte Aufgaben

  • So wird eine Ressourcenadapterkonfiguration erstellt

  • So wird eine Ressourcenadapterkonfiguration bearbeitet

  • So wird eine Ressourcenadapterkonfiguration gelöscht

Zugehörige asadmin-Befehle

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)

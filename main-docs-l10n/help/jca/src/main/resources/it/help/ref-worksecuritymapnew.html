Nuova mappa di sicurezza lavoro

Utilizzare la pagina Nuova mappa di sicurezza lavoro per creare una nuova mappa di sicurezza lavoro.

La pagina Nuova mappa di sicurezza lavoro contiene le opzioni riportate di seguito.

Nome mappa di sicurezza

Il nome della mappa di sicurezza lavoro.

Gruppo

Se il pulsante di scelta Mapping gruppi è selezionato, nel campo di testo deve essere specificata una mappa di ciascun gruppo di utenti EIS backend al gruppo di utenti corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se il pulsante di scelta Mapping principal è selezionato, nel campo di testo deve essere specificata la mappa di ciascun principal EIS backend al principal corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Resource Adapter

Il Resource Adapter per il sistema EIS.

Descrizione

La descrizione della mappa di sicurezza lavoro.

Stato

Se questa opzione è selezionata, la mappa di sicurezza lavoro è abilitata. Questa opzione è abilitata per impostazione predefinita.

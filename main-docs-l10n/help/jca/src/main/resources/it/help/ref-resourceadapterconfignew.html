Nuova configurazione Resource Adapter

Utilizzare la pagina Nuova configurazione Resource Adapter per creare una nuova configurazione di Resource Adapter.

La pagina Nuova configurazione Resource Adapter contiene le opzioni riportate di seguito.

Nome Resource Adapter

Il nome del Resource Adapter che si sta configurando.

ID pool di thread

Il pool di thread da usare per la configurazione di Resource Adapter.

Proprietà aggiuntive

Proprietà aggiuntive per il Resource Adapter. Le proprietà disponibili sono i nomi dei metodi setter della classe a cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome della classe del componente JavaBeans del Resource Adapter. Qualsiasi modifica apportata alle proprietà qui definite sostituisce i valori predefiniti presenti nel file ra.xml.

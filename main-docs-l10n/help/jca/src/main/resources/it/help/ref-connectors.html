Connettori

Utilizzare la pagina Connettori per configurare i tipi di risorse connettore riportati di seguito.

  • Risorse connettore

  • Connection pool connettore

  • Risorse oggetto amministrato

  • Mappe di sicurezza lavoro

Un modulo connettore, denominato anche Resource Adapter, è un componente Java che consente l'interazione delle applicazioni con il software EIS (Enterprise Information System). Il software EIS include vari tipi di sistemi, tra cui ERP (Enterprise Resource Planning), elaborazione di transazioni mainframe, database non relazionali e servizi di messaggistica, quali JMS (Java Message Service).

Un connection pool connettore è un gruppo di connessioni riutilizzabili per un determinato sistema EIS.

Una risorsa connettore è un oggetto di programma che fornisce una connessione a un sistema EIS a un'applicazione.

Una risorsa oggetto amministrato fornisce una funzionalità specializzata per un'applicazione.

Una mappa di sicurezza lavoro esegue il mapping tra un principal associato a un'istanza di lavoro in entrata e un principal nel dominio di sicurezza di GlassFish Server.

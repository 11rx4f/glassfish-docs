新建工作安全性對應

使用「新建工作安全性對應」頁面可建立新的工作安全性對應。

「新建工作安全性對應」包含下列選項。

安全性對應名稱

工作安全性對應的名稱。

群組

若選取了「群組對應」單選按鈕，文字欄位必須指定每個後端 EIS 使用者群組與 GlassFish Server 安全性網域中定義的對應使用者群組的對應。使用以逗號分隔的清單指定一個以上的對應。例如︰

eis-group-1=server-group-1,eis-group-2=server-group-2
主體

若選取了「主體對應」單選按鈕，文字欄位必須指定每個後端 EIS 主體與 GlassFish Server 安全性網域中定義的對應主體的對應。使用以逗號分隔的清單指定一個以上的對應。例如︰

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
資源轉接器

EIS 的資源轉接器。

描述

工作安全性對應的描述。

狀態

若選取了此選項，則會啟用工作安全性對應。此選項預設為啟用。

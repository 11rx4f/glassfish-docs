Para Deletar uma Configuração do Adaptador de Recursos

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, selecione o nó Configurações do Adaptador de Recursos.

    A página Configurações do Adaptador de Recursos será aberta.

  3. Na página Configurações do Adaptador de Recursos, marque a caixa de seleção junto ao nome do config do adaptador de recursos que estiver deletando.

  4. Clique em Deletar.

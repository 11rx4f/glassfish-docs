Novo Recurso Conector

Use a página Novo Recurso Conector para criar um novo recurso do conector.

A página Novo Recurso Conector contém as seguintes opções.

Nome JNDI

Um nome exclusivo que identifica o recurso do conector.

Nome do Pool

O pool de conexões ao qual pertence o recurso do conector.

Descrição

Uma breve descrição do recurso do conector.

Status

Se esta opção for selecionada, o recurso do conector será ativado. Por default, a opção está ativada.

Propriedades Adicionais

Propriedades adicionais para o recurso conector. As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector. Os pares de nome e valor especificados nessa tabela podem ser usados para substituir os valores default definidos pelo fornecedor do adaptador de recurso.

Alvos

(Esta opção só será exibida se os clusters ou instâncias standalone tiverem sido criados no domínio.)

Clusters e instâncias standalone do recurso. Mova os alvos desejados para a coluna Alvos Selecionados utilizando os botões Adicionar e Adicionar Tudo. Mova os alvos desnecessários para a coluna Alvos Disponíveis utilizando os botões Remover e Remover Tudo.

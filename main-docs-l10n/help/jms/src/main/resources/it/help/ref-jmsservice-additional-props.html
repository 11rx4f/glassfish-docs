Proprietà specifiche del servizio JMS

Per il servizio JMS sono disponibili le proprietà di configurazione del broker Oracle Message Queue riportate di seguito.

instance-name

Specifica il nome di istanza completo del broker Message Queue. L'impostazione predefinita è imqbroker.

instance-name-suffix

Specifica il suffisso da aggiungere al nome di istanza completo del broker Message Queue. Il suffisso è separato dal nome di istanza tramite un carattere di sottolineatura (_). Ad esempio, se il nome dell'istanza è imqbroker, con l'aggiunta del suffisso xyz viene modificato in imqbroker_xyz.

append-version

Se l'impostazione è true, aggiunge al nome completo dell'istanza del broker Message Queue i numeri per la parte intera e decimale della versione, preceduti dai caratteri di sottolineatura (_). Ad esempio, se il nome dell'istanza è imqbroker, con l'aggiunta dei numeri di versione viene modificato in imqbroker_8_0. L'impostazione predefinita è false.

user-name

Il nome utente per la creazione della connessione JMS. Questo valore è necessario solo se il nome utente predefinito o la password predefinita guest/guest non è disponibile nel broker. Il valore predefinito è guest.

password

La password per la creazione della connessione JMS. Questo valore è necessario solo se il nome utente predefinito o la password predefinita guest/guest non è disponibile nel broker. Il valore predefinito è guest.

Per eliminare una destinazione fisica JMS

L'eliminazione della destinazione di sistema, mq.sys.dmq, è sconsigliata.

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Java Message Service.

  3. Nel nodo Java Message Service selezionare il nodo Destinazioni fisiche.

    Viene visualizzata la pagina Destinazioni fisiche.

  4. Nella pagina Destinazioni fisiche selezionare la casella di controllo in corrispondenza del nome della destinazione da eliminare.

  5. Fare clic su Elimina.

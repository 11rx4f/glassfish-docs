Per visualizzare le statistiche per una destinazione fisica JMS

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Java Message Service.

  3. Nel nodo Java Message Service selezionare il nodo Destinazioni fisiche.

    Viene visualizzata la pagina Destinazioni fisiche.

  4. Nella pagina Destinazioni fisiche fare clic sul collegamento Visualizza relativo alla destinazione di cui si desidera visualizzare le statistiche.

    Viene visualizzata la pagina Visualizza statistiche destinazioni fisiche JMS.

  5. Al termine della visualizzazione delle statistiche, fare clic sul pulsante Indietro.

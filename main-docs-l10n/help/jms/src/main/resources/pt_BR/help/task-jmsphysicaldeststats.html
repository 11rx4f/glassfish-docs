Para Exibir Estatísticas de um Destino Físico JMS

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó do Serviço JMS.

  3. No nó Serviço JMS, selecione o nó Destinos Físicos.

    A página Destinos Físicos é aberta.

  4. Na página Destinos Físicos, clique no link Exibir para o destino para o qual você está visualizando a estatística.

    A página Exibir Estatísticas de Destino Físico JMS é aberta.

  5. Quando finalizar a exibição da estatística, clique no botão Voltar.

Recursos de Destino JMS

Utilize a página Recursos de Destino JMS para configurar os recursos de destino JMS.

Para cada recurso de destino, as seguintes informações são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso de destino.

Ativado

(Esta coluna só será exibida se a instância do servidor default, server, existir no domínio.)

Exibe uma marca de seleção se o recurso estiver ativado ou um X se o recurso estiver desativado.

Status

(Esta coluna só será exibida se os clusters ou instâncias standalone tiverem sido criadas no domínio.)

O número de clusters e instâncias independentes ao qual o recurso está direcionado e em quantos desses o recurso está ativado. Por exemplo, "Ativado em 2 de 4 Alvos" significa que o recurso está direcionado para quatro clusters e instâncias standalone e que está ativado em dois desses quatro alvos.

Tipo de Recurso

O tipo de recurso do recurso de destino.

Descrição

Uma descrição do recurso de destino.

O quadro Recursos de Destino também contém as opções a seguir.

Novo

Botão utilizado para criar um novo recurso de destino.

Deletar

Botão utilizado para deletar um ou mais recursos de destino selecionados.

Ativar

Botão utilizado para ativar um ou mais recursos de destino selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos de destino selecionados.

Suppression d'une destination physique JMS

La suppression de la destination système mq.sys.dmq n'est pas recommandée.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Service JMS.

  3. Sous le noeud Service JMS, sélectionnez le noeud Destinations physiques.

    La page Destinations physiques apparaît.

  4. Sur la page Destinations physiques, sélectionnez la case en regard du nom de la destination à supprimer.

  5. Cliquez sur Supprimer.

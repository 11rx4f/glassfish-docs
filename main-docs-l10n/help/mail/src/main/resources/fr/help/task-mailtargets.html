Gestion des cibles de session Jakarta Mail

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez Sessions Jakarta Mail.

    La page Sessions Jakarta Mail apparaît.

  3. Dans la table Sessions, cliquez sur le nom de session dont vous souhaitez gérer les cibles.

    La page Modifier une session Jakarta Mail apparaît.

  4. Cliquez sur l'onglet Cible.

    La page Cibles de sessions Jakarta Mail apparaît.

  5. Dans la table Cibles, sélectionnez l'action que vous souhaitez effectuer :

    • Pour ajouter ou enlever des cibles, cliquez sur Gérer les cibles.

    • Pour activer une cible, cochez la case située en regard de l'instance de serveur ou du cluster de votre choix, puis cliquez sur Activer.

    • Pour désactiver une cible, cochez la case située en regard de l'instance de serveur ou du cluster de votre choix, puis cliquez sur Désactiver.

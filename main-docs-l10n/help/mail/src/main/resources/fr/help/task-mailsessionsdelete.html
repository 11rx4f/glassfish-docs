Suppression d'une session Jakarta Mail

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez Sessions Jakarta Mail.

    La page Sessions Jakarta Mail apparaît.

  3. Sur la page Sessions Jakarta Mail, cochez la case située en regard du nom de la session à supprimer.

  4. Cliquez sur Supprimer.

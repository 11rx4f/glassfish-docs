Per eliminare una sessione Jakarta Mail

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare Sessioni Jakarta Mail.

    Viene visualizzata la pagina Sessioni Jakarta Mail.

  3. Nella pagina Sessioni Jakarta Mail selezionare la casella di controllo in corrispondenza del nome della sessione da eliminare.

  4. Fare clic su Elimina.

So wird eine Jakarta Mail-Session gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "Jakarta Mail-Sessions" aus.

    Die Seite "Jakarta Mail-Sessions" wird geöffnet.

  3. Aktivieren Sie auf der Seite "Jakarta Mail-Sessions" das Kontrollkästchen neben dem Namen der zu löschenden Session.

  4. Klicken Sie auf "Löschen".

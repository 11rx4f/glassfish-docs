Para ver ejecuciones de trabajo por lotes

  1. En el árbol de navegación, seleccione el cluster o la instancia de servidor de GlassFish Server para el que desea ver las ejecuciones de trabajo por lotes.

    Se abrirá la página Información General.

  2. Seleccione el separador Lote.

    Se abrirá la página Ejecuciones de trabajo por lotes.

  3. En la página Ejecuciones de trabajo por lotes, haga clic en un identificador de ejecución para ver los detalles sobre esa ejecución concreta.

    Se abrirá la página Detalles de ejecución de trabajo por lotes.

  4. Para ver detalles sobre los pasos de la ejecución de de trabajo por lotes, haga clic en el separador Pasos de ejecución.

    Se abrirá la página Pasos de ejecución de trabajo por lotes.

  5. Haga clic en el botón Atrás para volver a la página Ejecuciones de trabajo por lotes.

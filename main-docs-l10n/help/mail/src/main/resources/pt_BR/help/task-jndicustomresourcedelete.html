Para Deletar um Recurso JNDI Personalizado

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JNDI.

  3. No nó JNDI, selecione Recursos Personalizados.

    A página Recursos Personalizados é aberta.

  4. Na página Recursos Personalizados, marque a caixa de seleção ao lado do nome do recurso a ser deletado.

  5. Clique em Deletar.

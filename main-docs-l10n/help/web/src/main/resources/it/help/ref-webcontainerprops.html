Proprietà specifiche del contenitore Web

Per il contenitore Web è disponibile la proprietà generale aggiuntiva riportata di seguito.

dispatcher-max-depth

Evita che le istruzioni include o forward ricorsive creino un loop infinito mediante l'impostazione di un livello massimo di dispatch nidificati. Se questo livello viene superato, nel log del server viene scritto il messaggio seguente:

Exceeded maximum depth for nested request dispatches

Il valore predefinito è 20.

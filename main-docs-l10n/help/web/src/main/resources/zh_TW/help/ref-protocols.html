協定

使用「協定」頁面可組態網路協定。

對於每個協定，均提供以下資訊。

名稱

協定的名稱。

啟用安全性

如果已啟用協定的安全性，則為 True；如果未啟用協定的安全性，則為 False。

協定表也包含下列選項。

新建

建立新協定的按鈕。

刪除

刪除一或多個選取協定的按鈕。

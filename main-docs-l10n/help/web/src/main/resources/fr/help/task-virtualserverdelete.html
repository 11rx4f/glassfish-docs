Suppression d'un serveur virtuel

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sélectionnez la configuration à modifier.

  3. Sous le noeud Configuration, sélectionnez le noeud Serveurs virtuels.

    La page Serveurs virtuels apparaît.

  4. Sur la page Serveurs virtuels, cochez la case située à gauche du nom du serveur virtuel que vous supprimez.

  5. Cliquez sur Supprimer.

    La suppression de l'un des serveurs virtuels par défaut (__asadmin et server) est déconseillée.

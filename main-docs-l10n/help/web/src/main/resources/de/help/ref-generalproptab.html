Allgemeine Eigenschaften

Verwenden Sie die Seite "Allgemeine Eigenschaften", um Eigenschaften für den Webcontainer hinzuzufügen oder zu ändern.

Der Webcontainer hostet Webanwendungen. Er stellt die Umgebung bereit, in der Servlets und JavaServer Pages-(JSP-)Dateien ausgeführt werden. Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Für den Webcontainer spezifische Eigenschaften.

Für jede Eigenschaft enthält die Seite "Allgemeine Eigenschaften" die folgenden Informationen.

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft.

Beschreibung

Eine Beschreibung der Eigenschaft.

Verwandte Aufgaben

  • So werden allgemeine Eigenschaften für den Webcontainer konfiguriert

  • So werden Sessioneigenschaften für den Webcontainer konfiguriert

  • So werden Managereigenschaften für den Webcontainer konfiguriert

  • So werden Speichereigenschaften für den Webcontainer konfiguriert

  • So werden Verfügbarkeitseinstellungen für den Webcontainer konfiguriert

Propriedades Específicas para o Contêiner Web

As seguintes propriedades gerais adicionais estão disponíveis para o contêiner Web.

dispatcher-max-depth

Impede que as instruções recursivas de inclusão ou encaminhamento criem um loop infinito, pois define um nível máximo de distribuição aninhada. Se esse nível for excedido, a seguinte mensagem será gravada no log de servidor:

Exceeded maximum depth for nested request dispatches

O valor default é 20.

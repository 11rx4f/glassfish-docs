Para Deletar um Transporte

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó Configuração da Rede.

  3. No nó Config da Rede, selecione o nó Transportes.

    A página Transportes será aberta.

  4. Na página Transportes, marque a caixa de seleção à esquerda do transporte que estiver deletando.

  5. Clique em Deletar.

Propiedades Generales

Utilice la página Propiedades Generales para agregar o modificar propiedades del contenedor web.

El contenedor web aloja aplicaciones web. Proporciona el entorno en el que se ejecutan los servlets y los archivos de JavaServer Pages (JSP). Para obtener una descripción de las propiedades disponibles, consulte Propiedades específicas del contenedor web.

La página Propiedades Generales contiene la siguiente información para cada propiedad.

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad.

Descripción

Descripción de la propiedad.

Para suprimir un servidor virtual

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración que desea cambiar.

  3. En el nodo Configuración, seleccione el nodo Servidores Virtuales.

    Se abrirá la página Servidores Virtuales.

  4. En la página Servidores Virtuales, active la casilla de control situada a la izquierda del nombre del servidor virtual que va a suprimir.

  5. Haga clic en Suprimir.

    No se recomienda suprimir ninguno de los servidores por defecto __asadmin ni server.

Para suprimir un pool de threads

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, seleccione el nodo Pools de Threads.

    Se abrirá la página Pools de Threads.

  3. En la página Pools de Threads, active la casilla de control situada a la izquierda del pool de threads que se va a suprimir.

  4. Haga clic en Suprimir.

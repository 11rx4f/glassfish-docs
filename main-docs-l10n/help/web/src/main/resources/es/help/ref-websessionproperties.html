Propiedades específicas de sesiones de contenedor web

Las siguientes propiedades adicionales están disponibles para una sesión del contenedor web.

enableCookies

Si se define en true, utiliza cookies para el seguimiento de sesión. El valor por defecto es true.

enableURLRewriting

Permite reescribir la URL. Esto proporciona el seguimiento de la sesión por medio de la reescritura de la URL cuando el explorador no acepta cookies. También puede utilizar una llamada a encodeURL o a encodeRedirectURL en el servlet o en la página JavaServer Pages (JSP). El valor por defecto es true.

idLengthBytes

Especifica el número de bytes en este ID de sesión del módulo web. El valor por defecto es 128.

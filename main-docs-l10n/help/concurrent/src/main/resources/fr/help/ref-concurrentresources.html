Ressources simultanées

Utilisez les pages Ressources simultanées pour configurer les ressources simultanées des types suivants :

  • Services de contexte

  • Fabriques de threads gérés

  • Services exécuteurs gérés

  • Services exécuteurs planifiés gérés

Les ressources simultanées sont des objets gérés qui fournissent des fonctions d'accès simultané aux applications Java EE comme le définit la spécification JSR 236 sur les utilitaires d'accès simultané pour Java EE.

Dans GlassFish Server, configurez des ressources simultanées et rendez-les disponibles pour utilisation par les composants d'application tels que les servlets et les EJB. Les ressources simultanées sont accessibles via la recherche JNDI ou l'injection de ressource.

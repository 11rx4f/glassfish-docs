Recursos simultáneos

Utilice las páginas de recursos simultáneos para configurar recursos simultáneos de los siguientes tipos:

  • Servicios de contexto

  • Fábricas de threads gestionados

  • Servicios de ejecutor gestionados

  • Servicios de ejecutor programados gestionados

Los recursos simultáneos son objetos gestionados que proporcionan capacidades de simultaneidad a las aplicaciones de Jakarta EE como se define en la especificación JSR 236, sobre utilidades de simultaneidad para Jakarta EE.

En GlassFish Server, puede configurar recursos simultáneos y ponerlos a disposición de componentes de aplicación, como servlets y EJB. A los recursos simultáneos se accede a través de una consulta de JNDI o de inyección de recursos.

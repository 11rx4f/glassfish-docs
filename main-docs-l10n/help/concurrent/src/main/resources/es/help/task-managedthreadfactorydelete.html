Para suprimir una fábrica de threads gestionada

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos simultáneos.

  3. En el nodo Recursos simultáneos, seleccione Fábricas de threads gestionadas.

    Se abrirá la página Fábricas de threads gestionadas.

  4. En la página Fábricas de threads gestionadas, active la casilla de control situada junto al nombre del recurso que desee suprimir.

  5. Haga clic en Suprimir.

Recursos Concorrentes

Use as páginas Recursos Concorrentes para configurar recursos concorrentes dos seguintes tipos:

  • Serviços de Contexto

  • Fábricas de Threads Gerenciados

  • Serviços do Executor Gerenciado

  • Serviços do Executor Programado Gerenciado

Recursos simultâneos são objetos gerenciados que fornecem capacidades de simultaneidade para aplicações Jakarta EE conforme definido na especificação JSR 236, Concurrency Utilities for Jakarta EE.

NoGlassFish Server, você configura recursos simultâneos e os disponibiliza para uso por componentes do aplicativo como servlets e EJBs. Os recursos simultâneos são acessados por meio de consulta JNDI ou injeção de recurso.

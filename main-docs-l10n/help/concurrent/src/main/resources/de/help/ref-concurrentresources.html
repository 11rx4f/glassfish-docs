Nebenläufige Ressourcen

Auf den Seiten "Nebenläufige Ressourcen" können Sie die folgenden nebenläufigen Ressourcentypen konfigurieren:

  • Kontextservices

  • Managed Thread-Factorys

  • Managed Executor-Services

  • Verwaltete geplante ExecutorServices

Gleichzeitige Ressourcen sind verwaltete Objekte, die dafür sorgen, dass Java EE-Anwendungen wie in der JSR 236-Spezifikation "Concurrency Utilities for Java EE" definiert Aufgaben parallel ausführen können.

In GlassFish Server konfigurieren Sie gleichzeitige Ressourcen und stellen sie für Anwendungskomponenten wie Servlets und EJBs bereit. Auf gleichzeitige Ressourcen wird per JNDI-Lookup oder per Ressourcen-Injection zugegriffen.

Per eliminare una sessione JavaMail

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare Sessioni JavaMail.

    Viene visualizzata la pagina Sessioni JavaMail.

  3. Nella pagina Sessioni JavaMail selezionare la casella di controllo in corrispondenza del nome della sessione da eliminare.

  4. Fare clic su Elimina.

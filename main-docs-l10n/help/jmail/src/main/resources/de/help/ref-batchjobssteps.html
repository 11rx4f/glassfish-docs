Schritte der Batchjobausführung

Auf der Seite "Schritte der Batchjobausführung" werden Details zu den Schritten in einer bestimmten Batchjobausführung angezeigt. Es werden folgende Informationen angezeigt.

Instanzname
Clustername

Der Name des GlassFish-Clusters oder der GlassFish-Serverinstanz, auf das bzw. auf die sich die Konfiguration bezieht.

Jobname

Der Name des Batchjobs.

Ausführungs-ID

Die ID, die der Ausführung des Batchjobs zugewiesen ist. Beim erstmaligen Starten eines Jobs und bei jedem Neustart der vorhandenen Ausführung wird eine neue Ausführung erstellt.

Jobschritte

Die Schritte in einer bestimmten Batchjobausführung. Für jeden Schritt werden die folgenden Informationen angezeigt:

Schrittname

Der Name des Schritts.

Batchstatus

Der von der Batchlaufzeit festgelegte Status des Schritts.

Beendigungsstatus

Der Status des Schritts, der von der Job-XML für den Job oder von der Batchanwendung festgelegt wurde. Standardmäßig sind Beendigungsstatus und Batchstatus identisch, es sei denn, der Beendigungsstatus wird explizit überschrieben.

Startzeit

Der Zeitpunkt, an dem der Schritt gestartet wurde.

Endzeit

Der Zeitpunkt, an dem der Schritt abgeschlossen wurde.

Schrittmetriken

Metriken für den Schritt, die die Anzahl der gelesenen, geschriebenen, festgeschriebenen und weiteren Objekte auflisten.

Verwandte Aufgaben

  • So zeigen Sie Batchjobausführungen an

  • So konfigurieren Sie die Batchlaufzeit

Nuevo Recurso Externo

Utilice la página Nuevo recurso externo para crear un recurso JNDI externo.

La página Nuevo recurso externo contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso.

Tipo de recurso

Definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

Nombre especificado por el usuario para la clase de fábrica. Esta clase implanta la interfaz javax.naming.spi.InitialContextFactory.

Búsqueda JNDI

El valor JNDI para buscar en el repositorio externo.

Por ejemplo, cuando se crea un recurso externo para conectar a un repositorio externo, para probar una clase de bean, la búsqueda JNDI puede seguir el formato: cn=testmybean.

Descripción

Descripción de este recurso externo.

Estado

Estado del recurso externo. El recurso externo puede activarse o desactivarse. El valor por defecto es Activado.

Propiedades Adicionales

Propiedades adicionales del recurso externo. GlassFish Server no define ninguna propiedad adicional para los recursos externos.

Destinos

(Esta opción sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Clusters e instancias independientes para el recurso. Mueva los destinos seleccionados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.

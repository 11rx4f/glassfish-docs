Recursos Externos

Um recurso JNDI externo acessa um repositório JNDI externo, como um servidor LDAP.

Use a página Recursos Externos para configurar recursos JNDI externos.

Para cada recurso externo, as seguintes informações são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso externo.

Ativado

(Esta coluna só será exibida se a instância do servidor default, server, existir no domínio.)

Exibe uma marca de seleção se o recurso estiver ativado ou um X se o recurso estiver desativado.

Status

(Esta coluna só será exibida se os clusters ou instâncias standalone tiverem sido criadas no domínio.)

O número de clusters e instâncias independentes ao qual o recurso está direcionado e em quantos desses o recurso está ativado. Por exemplo, "Ativado em 2 de 4 Alvos" significa que o recurso está direcionado para quatro clusters e instâncias standalone e que está ativado em dois desses quatro alvos.

Tipo de Recurso

O tipo do recurso externo.

Descrição

Uma descrição do recurso externo.

A tabela Recursos também contém as opções a seguir.

Novo

Botão para criar um novo recurso externo.

Deletar

Botão para deletar um ou mais recursos externos selecionados.

Ativar

Botão para ativar um ou mais recursos externos selecionados.

Desativar

Botão para desativar um ou mais recursos externos selecionados.

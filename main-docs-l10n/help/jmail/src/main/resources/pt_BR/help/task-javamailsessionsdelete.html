Para Deletar uma Sessão JavaMail

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, selecione Sessões JavaMail.

    A página Sessões JavaMail é aberta.

  3. Na página Sessões JavaMail, marque a caixa de seleção ao lado do nome da sessão a ser deletada.

  4. Clique em Deletar.

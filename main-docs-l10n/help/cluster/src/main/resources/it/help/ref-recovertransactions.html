Recupera transazioni

Utilizzare la pagina Recupera transazioni per recuperare le transazioni per un'istanza di GlassFish Server.

Se l'istanza è in esecuzione, la pagina Recupera transazioni contiene solo il testo Il recupero viene eseguito dallo stesso server in esecuzione e il pulsante OK.

Se l'istanza non è in esecuzione, la pagina Recupera transazioni contiene le opzioni riportate di seguito.

Nome istanza

Il nome assegnato all'istanza quando è stata creata. Questo campo è di sola lettura.

Server di destinazione

Un'altra istanza in grado di recuperare le transazioni nell'istanza corrente.

Directory log delle transazioni

La directory in cui sono memorizzati i log delle transazioni. L'impostazione predefinita è /logs/tx nella directory del dominio.

I log delle transazioni vengono conservati nella sottodirectory tx della directory specificata dall'impostazione Posizione log delle transazioni del servizio transazioni per la configurazione del cluster. Il valore Posizione log delle transazioni predefinito è la directory specificata nel campo Radice log della pagina Attributi dominio, situata nella scheda Avanzate del nodo GlassFish Server.

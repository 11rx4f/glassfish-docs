Cluster

Nella pagina Cluster viene visualizzata la lista di cluster del dominio. Per ciascun cluster vengono visualizzate le informazioni riportate di seguito.

Nome

Il nome assegnato al cluster al momento della creazione del cluster. Se si fa clic sul nome, viene visualizzata la pagina Informazioni generali per il cluster.

Configurazione

La configurazione selezionata per il cluster al momento della creazione del cluster. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurazioni per il cluster.

Istanze

Il nome e lo stato di ogni istanza del cluster. Se si fa clic sul nome dell'istanza, viene visualizzata la pagina Informazioni generali per l'istanza.

La tabella Cluster contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un cluster.

Elimina

Pulsante che consente di eliminare un cluster.

Avvia cluster

Pulsante che consente di avviare un cluster.

Arresta cluster

Pulsante che consente di arrestare un cluster.

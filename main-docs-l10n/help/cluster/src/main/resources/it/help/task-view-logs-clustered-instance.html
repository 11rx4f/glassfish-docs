Per visualizzare i file di log per un'istanza di GlassFish Server in cluster

Prima di iniziare

È necessario che l'istanza per la quale si visualizzano i file di log sia in esecuzione.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale visualizzare i file di log.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic su Visualizza file di log.

    Viene visualizzata la finestra Visualizzatore log.

    Nota:

    L'istanza deve essere già in esecuzione. Se l'istanza non è in esecuzione, il pulsante Visualizza file di log è disattivato.

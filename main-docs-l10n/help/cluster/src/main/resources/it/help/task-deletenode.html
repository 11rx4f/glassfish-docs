Per eliminare i nodi

L'eliminazione comporta la rimozione del nodo dalla configurazione del server di amministrazione del dominio (DAS). Le directory e i file del nodo vengono eliminati quando viene eliminata l'ultima istanza di GlassFish Server residente nel nodo.

È inoltre possibile scegliere di disinstallare il software di GlassFish Server dall'host del nodo all'eliminazione del nodo.

Prima di iniziare

Assicurarsi che il nodo da eliminare non contenga istanze di GlassFish Server.

  1. Selezionare il nodo Nodi nell'albero di navigazione.

    Viene visualizzata la pagina Nodi.

  2. Per ogni nodo da eliminare, selezionare la casella di controllo in corrispondenza del nodo nella tabella Nodi.

    Nota:

    Non è possibile eliminare il nodo predefinito localhost-domain.

  3. Fare clic sul pulsante per eliminare i nodi oppure eliminare i nodi e disinstallare il software di GlassFish Server dagli host del nodo.

    • Per eliminare i nodi senza disinstallare il software di GlassFish Server, fare clic su Elimina.

      Viene richiesto di confermare l'eliminazione dei nodi.

    • Per eliminare i nodi e disinstallare il software di GlassFish Server, fare clic su Elimina e disinstalla.

      Viene richiesto di confermare l'eliminazione dei nodi e la disinstallazione del software di GlassFish Server dagli host del nodo.

      Nota:

      Il software di GlassFish Server non viene disinstallato da alcun host rappresentato da un nodo di tipo CONFIG. Viene eliminato solo il nodo.

  4. Confermare che si desidera eliminare i nodi o eliminare i nodi e disinstallare il software di GlassFish Server dagli host del nodo.

    I nodi vengono eliminati e rimossi dalla tabella dei nodi della pagina Nodi. Se si è fatto clic su Elimina e disinstalla, viene eseguita anche la disinstallazione del software di GlassFish Server dagli host del nodo.

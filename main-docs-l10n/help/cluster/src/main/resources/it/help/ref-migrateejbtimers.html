Esegui migrazione timer EJB

La pagina Esegui migrazione timer EJB contiene le opzioni riportate di seguito.

Origine

Elenco a discesa delle istanze di GlassFish Server arrestate nel cluster. La migrazione dei timer EJB può essere eseguita solo da istanze non in esecuzione.

Destinazione

Elenco a discesa delle istanze di GlassFish Server in esecuzione nel cluster. La migrazione dei timer EJB può essere eseguita solo verso istanze in esecuzione.

Nota:

Se tutte le istanze nel cluster non sono in esecuzione, nella pagina viene visualizzata solo un'avvertenza che indica l'indisponibilità di istanze in esecuzione verso le quali eseguire la migrazione dei timer EJB.

La pagina Esegui migrazione timer EJB contiene una tabella di timer EJB con le informazioni riportate di seguito.

Nome server

Lista delle istanze di GlassFish Server nel cluster.

Conteggio timer

Il numero di timer persistenti in ciascuna istanza.

Task correlati

Per eseguire la migrazione dei timer EJB

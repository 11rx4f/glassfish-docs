列出已部署到集群的应用程序

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“应用程序”选项卡。

    此时将打开集群的“应用程序”页。

So werden Knoten gelöscht

Durch Löschen eines Knotens wird der Knoten aus der Konfiguration des Domainadministrationsservers (DAS, Domain Administration Server) entfernt. Die Verzeichnisse und Dateien des Knotens werden gelöscht, wenn die letzte GlassFish Server-Instanz auf dem Knoten gelöscht wird.

Sie können auch die GlassFish Server-Software von dem Knotenhost deinstallieren, wenn der Knoten gelöscht wird.

Bevor Sie beginnen

Vergewissern Sie sich, dass sich keine GlassFish Server-Instanzen auf dem zu löschenden Knoten befinden.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Knoten".

    Die Seite "Knoten" wird geöffnet.

  2. Aktivieren Sie für jeden zu löschenden Knoten das Kontrollkästchen neben dem Knoten in der Tabelle "Knoten".

    Hinweis:

    Der vordefinierte Knoten localhost-domain kann nicht gelöscht werden.

  3. Klicken Sie auf die Schaltfläche zum Löschen der Knoten, oder löschen Sie die Knoten und deinstallieren Sie die GlassFish Server-Software von den Knotenhosts.

    • Um die Knoten zu löschen, ohne die GlassFish Server-Software zu deinstallieren, klicken Sie auf "Löschen".

      Sie müssen das Löschen der Knoten bestätigen.

    • Um die Knoten zu löschen und die GlassFish Server-Software von den Knotenhosts zu deinstallieren, klicken Sie auf "Löschen und deinstallieren".

      Sie müssen bestätigen, dass Sie die Knoten löschen und die GlassFish Server-Software von den Knotenhosts deinstallieren möchten.

      Hinweis:

      Die GlassFish Server-Software wird nicht von einem Host deinstalliert, den ein Knoten des Typs CONFIG darstellt. Nur der Knoten wird gelöscht.

  4. Sie müssen bestätigen, dass Sie die Knoten löschen oder die Knoten löschen und die GlassFish Server-Software von den Knotenhosts deinstallieren möchten.

    Die Knoten werden gelöscht und aus der Tabelle der Knoten auf der Seite "Knoten" entfernt. Wenn Sie auf "Löschen und deinstallieren" geklickt haben, wird die GlassFish Server-Software ebenfalls von den Knotenhosts deinstalliert.

So werden Ressourcenüberwachungsdaten für eine Standalone-GlassFish Server-Instanz angezeigt

Bevor Sie beginnen

Vor dem Anzeigen von Überwachungsdaten müssen Sie den Überwachungsservice konfigurieren.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Standalone-Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die Sie Überwachungsdaten der Ressource anzeigen möchten.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  4. Klicken Sie auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcenüberwachung" wird geöffnet.

  5. Wählen Sie aus der Dropdown-Liste "Ansicht" die Ressource, für die Sie Überwachungsdaten anzeigen möchten.

  6. Klicken Sie auf "Aktualisieren", um die Ansicht der Überwachungsdaten zu aktualisieren.

Migrer les horloges EJB

La page Migrer les horloges EJB contient les options ci-après.

Source

Liste déroulante des instances GlassFish Server arrêtées dans le cluster. Vous pouvez migrer les horloges EJB uniquement à partir d'instances qui ne sont pas en cours d'exécution.

Destination

Liste déroulante des instances GlassFish Server en cours d'exécution dans le cluster. Vous ne pouvez migrer les horloges EJB que vers des instances en cours d'exécution.

Remarque:

Si toutes les instances du cluster sont arrêtées, cette page affiche uniquement un avertissement indiquant qu'aucune instance en cours d'exécution n'est disponible pour la migration des horloges EJB.

La page Migrer les horloges EJB contient une table des horloges EJB présentant les informations ci-après.

Nom du serveur

Liste des instances GlassFish Server dans le cluster.

Nombre d'horloges

Nombre d'horloges persistantes sur chaque instance.

Propriedades de Sistema do Cluster

A página Propriedades de Sistema do Cluster exibe uma lista das propriedades do sistema Java que estão definidas para o cluster selecionado. As propriedades do sistema Java são especificadas para o acionador da aplicação Java por meio da opção -D do acionador da aplicação Java quando GlassFish Server for iniciado.

O campo Nome do Cluster é um campo somente para leitura que exibe o nome do cluster selecionado.

São exibidas as seguintes informações para cada propriedade:

Nome da Variável da Instância

O nome da propriedade do sistema.

Valor Atual

Valor atualmente definido para a propriedade. Este campo é somente para leitura.

Valor Substituto

O valor com o qual a propriedade será definida depois que as alterações forem salvas.

A tabela Propriedades Adicionais também contém as seguintes opções.

Adicionar Propriedade

Botão para adicionar uma propriedade. Clicar neste botão adiciona uma linha à tabela Propriedades Adicionais.

Deletar Propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada retorna ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.

Serviço de Disponibilidade

Utilize a página Serviço de Disponibilidade para alterar as definições de disponibilidade de um cluster.

A página Serviço de Disponibilidade contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Serviço de Disponibilidade

Se a caixa de seleção Ativado estiver marcada, a disponibilidade está ativada para o cluster. Por default, essa opção está selecionada.

Propriedades Adicionais

Uma tabela de propriedades adicionais do Serviço de Disponibilidade.

O GlassFish Server não define propriedades adicionais do Serviço de Disponibilidade.

São exibidas as seguintes informações para cada propriedade:

Nome

O nome da propriedade.

Valor

Valor da propriedade definida para a instância selecionada.

Descrição

Uma descrição textual que fornece mais informações sobre a propriedade.

A tabela Propriedades Adicionais também contém as seguintes opções.

Adicionar Propriedade

Botão para adicionar uma propriedade. Clicar neste botão adiciona uma linha à tabela Propriedades Adicionais.

Deletar Propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada retorna ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.

Recuperar Transações

Utilize a página Recuperar Transações para recuperar transações de uma instância do GlassFish Server.

Se a instância estiver em execução, a página Recuperar Transações contém somente o texto A recuperação é realizada pelo próprio servidor em execução e o botão OK.

Se a instância não estiver em execução, a página Recuperar Transações contém as seguintes opções.

Nome da Instância

Nome designado à instância quando a instância foi criada. Este campo é somente para leitura.

Servidor de Destino

Outra instância que pode recuperar as transações na instância atual.

Diretório de Log de Transações

Diretório no qual os logs de transação são armazenados. O default é /logs/tx no diretório de domínio.

Os logs de transação são mantidos no subdiretório tx do diretório especificado pela definição do Localização do Log de Transações do Serviço de Transação na configuração do cluster. O valor default do Localização do Log de Transações é o diretório especificado pelo campo Raiz de Log da página Atributos do Domínio, que está localizado na guia Avançado do nó do GlassFish Server.

Migrar temporizadores EJB

La página Migrar temporizadores EJB contiene las opciones siguientes.

Origen

Lista desplegable de instancias de GlassFish Server paradas en el cluster. Los temporizadores EJB sólo se pueden migrar desde instancias que no se estén ejecutando.

Destino

Lista desplegable de instancias de GlassFish Server en ejecución en el cluster. Los temporizadores EJB sólo se pueden migrar a instancias que se estén ejecutando.

Nota:

Si no se está ejecutando ninguna instancia del cluster, esta página sólo muestra una advertencia que indica que no hay disponibles instancias en ejecución a las que se pueda migrar los temporizadores EJB.

La página Migrar temporizadores EJB contiene una tabla de temporizadores EJB con la información siguiente.

Nombre de servidor

Lista de instancias de GlassFish Server en el cluster.

Recuento de temporizadores

Número de temporizadores persistentes en cada instancia.

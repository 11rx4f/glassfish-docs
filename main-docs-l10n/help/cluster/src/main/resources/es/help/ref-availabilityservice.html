Servicio de disponibilidad

Utilice la página Servicio de disponibilidad para cambiar la configuración de disponibilidad para un cluster.

La página Servicio de disponibilidad contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Servicio de Disponibilidad

Si se activa la casilla de control Activado, se activa la disponibilidad para el cluster. Esta opción está activada por defecto.

Propiedades Adicionales

Tabla de propiedades adicionales para el servicio de disponibilidad.

GlassFish Server no define propiedades adicionales para el servicio de disponibilidad.

Para cada propiedad se muestra la información siguiente:

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad que se ha definido para la instancia seleccionada.

Descripción

Descripción textual que ofrece más información sobre la propiedad.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se agrega una fila a la tabla Propiedades adicionales.

Suprimir propiedades

Botón para suprimir una o varias propiedades seleccionadas. Cualquier propiedad que se suprima vuelve a su valor por defecto o, si no se ha definido ningún valor por defecto, se queda sin definir.

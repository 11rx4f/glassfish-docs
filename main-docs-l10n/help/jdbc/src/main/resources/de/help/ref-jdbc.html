JDBC

Auf der Seite "JDBC" können Sie die folgenden Ressourcentypen konfigurieren:

  • JDBC-Ressourcen

  • Connection-Pools

Die Java DataBase Connectivity-(JDBC-)Technologie stellt Konnektivität mit relationalen Datenbanken bereit. In GlassFish Server konfigurieren Sie Datenbankkonnektivität, indem Sie JDBC-Ressourcen (Datenquellen) zur Domain hinzufügen. Unter einer Datenquelle versteht man eine auf der Java-Plattform, Enterprise Edition (Jakarta EE-Plattform), basierende standardmäßige Methode zum Konfigurieren der Konnektivität mit einer Datenbank. Jede Datenquelle enthält einen Pool aus Datenbankverbindungen. Anwendungen suchen die Datenquelle in der Java Naming and Directory Interface-(JNDI-)API-Struktur oder im lokalen Anwendungskontext und rufen dann über die getConnection-Methode eine Datenbankverbindung ab. Datenquellen und die zugehörigen Connection-Pools stellen Prozesse zur Verbindungsverwaltung bereit, die den Systembetrieb aufrechterhalten.

GlassFish Server enthält eine Kopie der Java DB-Datenbank und ist für den Zugriff auf diese Datenbank vorkonfiguriert.

Per eliminare un connection pool JDBC

Quando si elimina un connection pool JDBC, vengono eliminate anche tutte le risorse JDBC che dipendono dal pool eliminato.

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo JDBC.

  3. Nel nodo JDBC selezionare il nodo Connection pool.

    Viene visualizzata la pagina Connection pool.

  4. Nella pagina Connection pool selezionare la casella di controllo a sinistra del nome del connection pool da eliminare.

  5. Fare clic su Elimina.

Logs de Domínio

Use a página Logs de Domínio para coletar e fazer download de arquivos de log para uma instância ou cluster especificados do GlassFish Server e compilar os logs em um arquivo compactado ZIP simples.

Observação:

Se você especificar um cluster, o arquivo compactado ZIP também incluirá o arquivo de log para o servidor de administração de domínios (DAS).

A página de Logs do Domínio contém a opções a seguir.

Cluster

Uma lista drop-down de clusters se existirem.

Instância

Uma lista drop-down de instâncias do servidor.

Coletar Logs

Botão para coletar os logs em um arquivo log-files-name.zip, em que name é o nome do cluster ou instância.

O arquivo log-files-name.zip é submetido a download usando o mecanismo de download de ’ do seu browser.

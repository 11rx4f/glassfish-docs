Recursos

Use a página Recursos para configurar os seguintes recursos para a instância ou cluster do &ProductName; selecionado(a).

  • JDBC

  • Conectores

  • Configs do Adaptador de Recurso

  • Recursos JMS

  • Sessões Jakarta Mail

  • JNDI

  • Recursos Concorrentes

A página Recursos também contém a seguinte opção:

Adicionar Recursos

Botão para criar um ou mais recursos de um arquivo glassfish-resources.xml.

Para Configurar os Recursos de GlassFish Server Para Usar pelos Provedores de Segurança da Mensagem

  1. Se estiver usando um token de nome de usuário, configure um banco de dados de usuário, se necessário.

    Ao usar um token de nome de usuário/senha, primeiro configure um realm apropriado e um banco de dados de usuário adequado para o realm.

    A configuração de um banco de dados do usuário é discutida em Para Editar um Realm.

  2. Gerencie os certificados e as chaves particulares, se necessário.

Próximas Etapas

Quando os recursos do GlassFish Server forem configurados para uso por provedores de segurança de mensagens, os provedores instalados com o GlassFish Server poderão ser ativados conforme descrito em Para Editar uma Configuração de Segurança de Mensagem.

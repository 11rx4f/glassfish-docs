Alvos do Recurso

Utilize a página Alvos do Recurso para exibir os clusters e as instâncias de servidores standalone do alvo nos quais o recurso pode ser ativado.

A página Alvos do Recurso contém as seguintes informações de cada alvo.

Nome do Alvo

Nome do cluster ou da instância de servidor standalone.

Ativado

Exibe uma marca de seleção se o recurso estiver ativado no alvo ou um X se o recurso estiver desativado.

A tabela Alvos também contém as seguintes opções.

Gerenciar Alvos

Botão para gerenciar os alvos do recurso.

Ativar

Botão para ativar o recurso no alvo selecionado.

Desativar

Ação para desativar o recurso no alvo selecionado.

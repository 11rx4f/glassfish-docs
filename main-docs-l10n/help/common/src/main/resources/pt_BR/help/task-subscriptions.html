Para Exibir a Página de Assinaturas de GlassFish Server

  1. Na árvore de navegação, selecione o nó Tarefas Comuns.

    A página Tarefas Comuns será aberta.

  2. Em Notícias do GlassFish, clique no link Assinaturas.

    A página Web Assinaturas do GlassFish é exibida em uma guia ou janela separada do browser.

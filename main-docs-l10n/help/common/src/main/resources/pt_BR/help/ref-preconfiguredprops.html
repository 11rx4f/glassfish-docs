Propriedades Predefinidas do Sistema

As propriedades do sistema a seguir são predefinidas para o GlassFish Server.

com.sun.aas.installRoot

O diretório em que o GlassFish Server está instalado.

com.sun.aas.instanceRoot

O diretório de nível superior de um domínio do GlassFish Server.

com.sun.aas.hostName

O nome do host (máquina).

com.sun.aas.javaRoot

O diretório de instalação do Java Runtime.

com.sun.aas.imqLib

O diretório da biblioteca do software Oracle Message Queue.

com.sun.aas.domainName

O nome do domínio. Esta propriedade não é usada na configuração default, mas pode ser usada para personalizar a configuração. O valor default é domain1.

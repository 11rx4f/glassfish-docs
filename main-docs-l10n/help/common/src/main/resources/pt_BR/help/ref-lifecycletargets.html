Alvos do Módulo do Ciclo de Vida

Utilize a página Alvos do Módulo do Ciclo de vida para exibir os clusters e as instâncias de servidores standalone nos quais o módulo do ciclo de vida pode ser ativado.

A página Alvos do Módulo do Ciclo de Vida contém as seguintes informações.

Nome do Alvo

Nome do cluster ou da instância de servidor standalone.

Ativado

Exibe true se o módulo do ciclo de vida estiver ativado no alvo ou false se o módulo do ciclo de vida estiver desativado.

A tabela Alvos também contém as seguintes opções.

Gerenciar Alvos

Botão para gerenciar os alvos do módulo do ciclo de vida.

Mais Ações

Lista drop-down das ações seguintes.

Ativar

Ação para ativar o módulo do ciclo de vida no alvo selecionado.

Desativar

Ação para desativar o módulo do ciclo de vida no alvo selecionado.

Tarefas Comuns

A página Tarefas Comuns fornece atalhos para as tarefas comuns do Console de Administração.

Use a página Tarefas Comuns para as seguintes operações:

  • Para Exibir a Página de Registro de GlassFish Server

  • Para Exibir a Página de Assinaturas de GlassFish Server

  • Para Exibir o Blog do GlassFish Server

  • Para Exibir uma Lista de Aplicações Implantadas

  • Para Implantar uma Aplicação Web

  • Para Implantar uma Aplicação Enterprise

  • Para Implantar um Cliente da Aplicação

  • Para Implantar um Módulo Conector

  • Para Implantar um EJB Jar

  • Para Configurar o Serviço de Monitoramento

  • Para Exibir Dados de Monitoramento da Aplicação

  • Para Exibir Dados de Monitoramento do Servidor

  • Para Exibir Dados de Monitoramento do Recurso

  • Para Exibir Componentes Instalados

  • Para Instalar Componentes Add-On

  • Para Exibir as Atualizações Disponíveis para os Componentes

  • Para Criar um Pool de Conexões JDBC

A página Tarefas Comuns também fornece links para alguns dos documentos do GlassFish Server

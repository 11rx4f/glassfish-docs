预定义系统属性

以下为 GlassFish Server 的预定义系统属性。

com.sun.aas.installRoot

安装 GlassFish Server 的目录。

com.sun.aas.instanceRoot

GlassFish Server 域的顶级目录。

com.sun.aas.hostName

主机 (计算机) 名称。

com.sun.aas.javaRoot

Java 运行时的安装目录。

com.sun.aas.imqLib

Oracle 消息队列 软件的库目录。

com.sun.aas.domainName

域名。在默认配置中，不使用此属性，但可以使用此属性来定制配置。默认值为 domain1。

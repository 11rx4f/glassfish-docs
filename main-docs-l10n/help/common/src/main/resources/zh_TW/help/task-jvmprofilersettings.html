組態 JVM 效能評測器設定

效能評測器工具產生用於分析效能和識別潛在瓶頸的資料。

  1. 在瀏覽樹狀結構中，選取「組態」節點。

  2. 選取您要變更的組態。

  3. 在「組態」節點下，選取「JVM 設定」節點。

    「JVM 一般設定」頁面會隨即開啟。

  4. 在「JVM 一般設定」頁面上，按一下「效能評測器」標籤。

    「JVM 效能評測器設定」頁面會隨即開啟。

  5. 指定效能評測器設定。

    您指定的資訊取決於所用之效能評測器產品。如需範例與說明，請參閱 Oracle GlassFish Server Application Development Guide 中的第 3 章 Debugging Applications。

  6. 按一下「儲存」。

  7. 重新啟動伺服器。

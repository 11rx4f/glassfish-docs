邏輯 JNDI 名稱

建立資源時，您需要指定資源的唯一 JNDI 名稱。應用程式將會使用此名稱來存取資源。

Jakarta EE 標準規定某些預設資源必須能夠供應用程式存取，而且必須替這些預設資源定義特定 JNDI 名稱。GlassFish Server 透過使用邏輯 JNDI 名稱 (會將 Jakarta EE 標準 JNDI 名稱對應至特定 GlassFish Server 資源) 提供這些名稱，如下所示：

java:comp/DefaultContextService

此 Jakarta EE 標準名稱會對應至 concurrent/__defaultContextService 資源。

java:comp/DefaultManagedThreadFactory

此 Jakarta EE 標準名稱會對應至 concurrent/__defaultManagedThreadFactory 資源。

java:comp/DefaultManagedExecutorService

此 Jakarta EE 標準名稱會對應至 concurrent/__defaultManagedExecutorService 資源。

java:comp/DefaultManagedScheduledExecutorService

此 Jakarta EE 標準名稱會對應至 concurrent/__defaultManagedScheduledExecutorService 資源。

java:comp/DefaultDataSource

此 Jakarta EE 標準名稱會對應至 jdbc/__default 資源。

java:comp/DefaultJMSConnectionFactory

此 Jakarta EE 標準名稱會對應至 jms/__defaultConnectionFactory 資源。

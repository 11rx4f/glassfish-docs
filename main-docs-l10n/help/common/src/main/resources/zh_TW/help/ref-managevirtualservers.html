虛擬伺服器目標

使用「虛擬伺服器目標」頁面可變更與應用程式或模組相關聯的目標虛擬伺服器。

「虛擬伺服器目標」頁面包含以下資訊。

可用的目標

與應用程式或模組無關聯的虛擬伺服器。

選取的目標

與應用程式或模組相關聯的虛擬伺服器。

增加

可將選取的目標從「可用的目標」移至「選取的目標」的按鈕。

Add All

可將所有「可用的目標」移至「選取的目標」的按鈕。

移除

可將選取的目標從「選取的目標」移至「可用的目標」的按鈕。

全部移除

可將所有「選取的目標」移至「可用的目標」的按鈕。

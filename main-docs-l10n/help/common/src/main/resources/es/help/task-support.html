Ver el blog de GlassFish Server

El blog de GlassFish Server ofrece noticias de la comunidad de GlassFish Server y enlaces a información adicional sobre dicho producto.

  1. En el árbol de navegación, seleccione el nodo Tareas comunes.

    Se abrirá la página Tareas comunes.

  2. En Noticias de GlassFish, haga clic en el enlace Blogs.

    El blog de GlassFish Server se abrirá en un separador o ventana del explorador independiente.

Editar un dominio

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee configurar los dominios.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

    Se abrirá la página Dominios.

  4. Seleccione el nombre de un dominio existente.

    Se abrirá la página Editar Dominio.

  5. Para modificar el nombre de clase del dominio, realice una de las siguientes acciones:

    • En la lista desplegable Nombre de Clase, seleccione el nombre de clase para el dominio creado.

    • Active el segundo botón de radio y escriba el nombre de clase del dominio personalizado creado.

  6. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    Las propiedades válidas dependen del tipo de dominio.

    • Propiedades específicas de la clase FileRealm

    • Propiedades específicas de la clase CertificateRealm

    • Propiedades específicas de la clase JDBCRealm

    • Propiedades específicas de la clase LDAPRealm

    • Propiedades específicas de la clase SolarisRealm

    • Propiedades específicas de la clase PamRealm

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  7. Haga clic en Guardar.

Agregar un usuario al dominio de administración

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee gestionar los usuarios del dominio de administración.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

  4. Seleccione el nodo admin-realm.

    Se abrirá la página Editar Dominio.

  5. En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

    Se abrirá la página Usuarios administradores.

  6. En la página Usuarios administradores, haga clic en Nuevo.

    Se abrirá la página Nuevo Usuario de Dominio de Archivo.

  7. En el campo ID de Usuario, introduzca el nombre del usuario.

  8. En el campo Lista de Grupos, introduzca la lista de los grupos a los que pertenece el usuario.

    Utilice comas para separar los nombres de grupo. Estos grupos no hace falta definirlos en otro lugar.

  9. En el campo Contraseña, introduzca una contraseña única.

  10. En el campo Confirmar Contraseña Nueva, escriba la contraseña de usuario de nuevo.

  11. Haga clic en Aceptar.

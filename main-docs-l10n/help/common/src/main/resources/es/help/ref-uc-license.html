Acuerdo de licencia

La página Acuerdo de licencia muestra los acuerdos de licencia de todos los componentes seleccionados para instalarse o actualizarse. El acuerdo de licencia de un componente lo proporciona el autor del componente con el mismo. Si existen varios componentes en el Repositorio de red, deben publicarse con licencias diferentes, por ejemplo:

  • La página Acuerdo de licencia contiene las siguientes opciones.

    Aceptar

    Este botón se utiliza para aceptar los términos de las licencias de todos los componentes seleccionados y seguir con la instalación o actualización.

    Rechazar

    Este botón se utiliza para rechazar los términos de las licencias de todos los componentes seleccionados y vuelve a la página en la que había seleccionado los componentes.

Supervisión de aplicaciones

Consulte la página Supervisión de Aplicación para ver los datos de supervisión de una aplicación seleccionada.

La página Supervisión de Aplicación contiene la siguiente información.

Nombre de instancia

Nombre de la instancia del servidor.

Aplicación

Aplicación para la que se muestran las estadísticas de supervisión.

Componente

Componente de la aplicación para el que se muestran las estadísticas de supervisión.

Para obtener una descripción de las propiedades supervisadas, consulte el capítulo 8, sobre la administración del servicio de supervisión, en Oracle GlassFish Server Administration Guide.

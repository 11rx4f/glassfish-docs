Configuración

Utilice el nodo de configuración para que una instancia o un cluster de GlassFish Server configure los componentes siguientes:

  • Configuración general de JVM

  • Configuración de Registrador

  • Contenedor Web

  • Contenedor EJB

  • Java Message Service

  • Seguridad

  • Servicio de Disponibilidad

  • Servicio de Transacción

  • Servicio HTTP

  • Servidores Virtuales

  • Configuración de Red

  • Pools de Threads

  • ORB

  • Servicio de administración

  • Supervisión

  • Servicio de Gestión de Grupos

  • Propiedades del sistema

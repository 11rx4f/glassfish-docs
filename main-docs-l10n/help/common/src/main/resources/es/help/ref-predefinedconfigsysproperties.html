Propiedades del sistema predefinidas

GlassFish Server define las siguientes propiedades del sistema de configuración:

ASADMIN_LISTENER_PORT

Esta propiedad especifica el número de puerto del puerto HTTP o HTTPS a través del cual el DAS se conecta a la instancia para gestionar la instancia. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

HTTP_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para recibir las solicitudes HTTP. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

HTTP_SSL_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para recibir solicitudes HTTPS. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

IIOP_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para las conexiones IIOP. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

IIOP_SSL_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para conexiones IIOP seguras. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

IIOP_SSL_MUTUALAUTH_PORT

Esta propiedad especifica el número del puerto que se utiliza para conexiones IIOP seguras con autenticación de cliente. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

JAVA_DEBUGGER_PORT

Esta propiedad especifica el número del puerto que se utiliza para las conexiones con el depurador de Java Platform Debugger Architecture (JPDA) (http://java.sun.com/javase/technologies/core/toolsapis/jpda/). Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

JMS_PROVIDER_PORT

Esta propiedad especifica el número de puerto del proveedor de Java Message Service. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

JMX_SYSTEM_CONNECTOR_PORT

Esta propiedad especifica el número del puerto en el que recibe el conector JMX. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

OSGI_SHELL_TELNET_PORT

Esta propiedad especifica el número del puerto utilizado para las conexiones con el Shell remoto Apache Felix (http://felix.apache.org/site/apache-felix-remote-shell.html). Este shell utiliza Felix Shell Service para interactuar con el subsistema de gestión de módulos OSGi. Los valores válidos están entre 1-65535. En sistemas UNIX, para la creación de sockets de recepción en los puertos 1-1024 se necesitan privilegios de superusuario.

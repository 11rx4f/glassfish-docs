Alias de Contraseña

Utilice la página Alias de Contraseña para crear y gestionar alias de contraseña.

La página Alias de Contraseña muestra una lista de los alias de contraseña del dominio. Para cada alias se muestra la información siguiente:

Nombre

Nombre asignado al alias de contraseña durante su creación. Al hacer clic en el nombre, se abrirá la página Editar alias de contraseña para el alias.

La tabla Alias de Contraseña también contiene las siguientes opciones.

Nuevo

Botón para crear un alias de contraseña.

Suprimir

Botón para suprimir uno más alias de contraseña seleccionados.

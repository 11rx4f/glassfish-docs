Agregar recursos configurados en un archivo XML

  1. En el árbol de navegación, seleccione el nodo Recursos.

    Se abrirá la página Recursos.

  2. Haga clic en el botón Agregar Recursos.

    Se abrirá la página Agregar Recursos.

  3. Seleccione una de las opciones siguientes para especificar la ubicación del archivo glassfish-resources.xml:

    Archivo XML que Se Va a Cargar en el Servidor

    Escriba la ruta de acceso del archivo si se encuentra en una máquina remota.

    Archivo XML local al que Se Puede Acceder desde GlassFish Server

    Escriba la ruta de acceso del archivo si se encuentra en la máquina del servidor de administración de dominio (DAS).

  4. En la lista desplegable Destino, seleccione el cluster o la instancia de servidor autónomo en la que desee crear los recursos.

  5. Haga clic en Aceptar.

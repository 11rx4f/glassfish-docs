Suprimir una configuración de proveedor de seguridad de mensajes

Antes de Empezar

No es posible crear o suprimir proveedores de seguridad de mensajes para la configuración por defecto server-config. Los únicos proveedores de seguridad de mensajes soportados con la configuración por defecto server-config son SOAP y HttpServlet. Dado que estos dos proveedores de mensajes se crean por defecto al instalar GlassFish Server, el botón Suprimir no aparece en la página Configuraciones de Seguridad de Mensajes para la configuración por defecto server-config.

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Amplíe el nodo Seguridad.

  3. Seleccione el nodo Seguridad de Mensajes.

    Se abrirá la página Configuraciones de Seguridad de Mensajes.

  4. En la página Configuraciones de Seguridad de Mensajes, seleccione SOAP.

    Se abrirá la página Editar Configuración de Seguridad de Mensajes.

  5. En la página Editar Configuración de Seguridad de Mensajes, haga clic en el separador Proveedores.

    Se abrirá la página Editar Configuración de Proveedor.

  6. En la página Editar Configuración de Proveedor, active la casilla de control que aparece a la izquierda de la configuración de proveedor que está suprimiendo.

  7. Haga clic en Suprimir.

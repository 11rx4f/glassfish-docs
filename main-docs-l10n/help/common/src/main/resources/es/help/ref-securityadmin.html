Administración Segura

Utilice la página Administración Segura para activar o desactivar la administración segura.

Cuando se activa la administración segura:

  • DAS y las instancias del dominio utilizan certificados SSL para cifrar los mensajes que se envían mutuamente.

  • DAS acepta mensajes de administración de los clientes del administrador remotos como la utilidad asadmin y los IDE.

La página Administración Segura contiene la información siguiente.

Activar Administración Segura

Botón para activar la administración segura. Este botón sólo aparece cuando la administración segura no está activada.

Nota:

Si algún usuario administrativo del dominio tiene una contraseña en blanco, los intentos de activar la administración segura fallan.
Desactivar Administración Segura

Botón para desactivar la administración segura. Este botón sólo aparece cuando la administración segura está activada.

Alias de Administración

Alias que hace referencia al certificado SSL/TLS en el DAS. Este alias lo utiliza el DAS para identificarse ante las instancias. El valor por defecto es s1as. Esta opción sólo aparece cuando la administración segura no está activada.

Alias de Instancia

Alias que hace referencia al certificado SSL/TLS en las instancias. Este alias lo utilizan las instancias para identificarse ante el DAS. El valor por defecto es glassfish-instance. Esta opción sólo aparece cuando la administración segura no está activada.

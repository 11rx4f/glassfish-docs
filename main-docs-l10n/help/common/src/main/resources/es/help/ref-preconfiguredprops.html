Propiedades del sistema predefinidas

En GlassFish Server existen las siguientes propiedades del sistema predefinidas.

com.sun.aas.installRoot

Directorio donde GlassFish Server está instalado.

com.sun.aas.instanceRoot

Directorio de nivel superior para un dominio GlassFish Server.

com.sun.aas.hostName

Nombre del host (máquina).

com.sun.aas.javaRoot

Directorio de instalación de Java Runtime.

com.sun.aas.imqLib

Directorio de biblioteca para el software Oracle Cola de mensajes.

com.sun.aas.domainName

Nombre del dominio. Esta propiedad no se utiliza en la configuración por defecto, pero se puede utilizar para personalizar la configuración. El valor por defecto es domain1.

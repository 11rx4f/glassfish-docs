Descriptores de modulo

Utilice la página Descriptores de módulo para ver los descriptores del módulo para una aplicación.

La página Descriptores de módulo contiene las siguientes opciones.

Descriptores de Despliegue

Tabla que muestra el nombre de aplicación, el del subcomponente (si procede) y el del archivo descriptor.

Propiedades del sistema

Consulte la página Propiedades del Sistema para ver y gestionar las propiedades de GlassFish Server.

Una propiedad del sistema define un valor común para una configuración en el nivel del servidor. Puede consultar una propiedad del sistema en un campo de texto de la consola de administración colocándola entre llaves y con un signo de dólar. Por ejemplo, para consultar una propiedad llamada prop-name, debe utilizar la sintaxis${prop-name}.

La página Propiedades del Sistema contiene la siguiente información para cada propiedad.

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad.

Descripción

Descripción opcional de la propiedad.

Existe un número de propiedades del sistema predefinidas pero no se muestran en esta página. Para obtener más información, consulte Propiedades del sistema predefinidas.

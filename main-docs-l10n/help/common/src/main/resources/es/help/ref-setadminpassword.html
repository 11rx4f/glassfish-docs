Nueva contraseña de administrador

Utilice la página Nueva Contraseña de Administrador para cambiar la contraseña de la cuenta de administrador actual.

Nota:

Si se activa la administración segura, no se puede cambiar la contraseña por un valor en blanco.

La página Nueva Contraseña de Administrador contiene las siguientes opciones.

ID de Usuario

ID de usuario del administrador actual. Éste es un campo de sólo lectura.

Lista de Grupos

Grupo al que pertenece el administrador. Éste es un campo de sólo lectura. El valor es asadmin.

Contraseña Nueva

Contraseña nueva del administrador.

Confirmar Contraseña Nueva

Contraseña nueva del administrador.

Recursos de Ámbito de Aplicación

Utilice la página Recursos de Ámbito de Aplicación para ver recursos disponibles sólo para una aplicación concreta.

La página Recursos de Ámbito de Aplicación contiene una tabla con la información siguiente.

Nombre de Aplicación

Nombre de la aplicación.

Nombre de Módulo

Nombre del módulo dentro de la aplicación.

Nombre de recurso

Nombre del recurso de ámbito de aplicación para la aplicación o el módulo. Haga clic en el nombre para modificar la configuración de los recursos.

Tipo de recurso

Tipo de recurso, por ejemplo Pools de Conexiones JDBC o Recursos de Conectores.

Ver la página de registro de GlassFish Server

La página Registro le permite registrar la instalación de GlassFish Server.

Como usuario registrado de GlassFish Server obtendrá ventajas como las siguientes:

  • Notificación de nuevas versiones, parches y actualizaciones

  • Ofertas especiales para formación, servicios y productos específicos para desarrolladores

  • Acceso a documentación técnica, artículos formativos y seminarios web con el equipo de producto de GlassFish

  1. En el árbol de navegación, seleccione el nodo Tareas comunes.

    Se abrirá la página Tareas comunes.

  2. En Noticias de GlassFish, haga clic en el enlace Registro.

    Se abrirá la página web Registro de GlassFish en un separador o ventana del explorador independiente.

사용권 계약

[사용권 계약 페이지]에는 설치 또는 업데이트를 위해 선택한 모든 구성 요소의 사용권 계약이 표시됩니다. 구성 요소의 사용권 계약은 구성 요소 작성자에 의해 구성 요소와 함께 제공됩니다. 네트워크 저장소의 다양한 구성 요소가 다양한 사용권 하에 게시될 수 있습니다. 예를 들면 다음과 같습니다.

[사용권 계약] 페이지에는 다음 옵션이 포함되어 있습니다.

동의

이 단추를 누르면 모든 선택한 구성 요소의 라이센스 조건에 동의하고 설치 또는 업그레이드를 진행합니다.

거부

이 단추를 누르면 모든 선택한 구성 요소의 라이센스 조건을 거부하고 구성 요소를 선택한 페이지로 돌아갑니다.

관련 작업

  • 구성 요소의 사용 가능한 업데이트를 확인하는 방법

  • 추가 기능 구성 요소를 설치하는 방법

Modifica realm

Utilizzare la pagina Modifica realm per modificare un realm.

La pagina Modifica realm contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome realm

Il nome del realm. Questo campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo realm.

Nome classe

Il nome di classe per il realm che si sta modificando.

Proprietà aggiuntive

Proprietà aggiuntive per il realm. Le proprietà valide dipendono dal tipo di realm.

  • Proprietà specifiche della classe FileRealm

  • Proprietà specifiche della classe CertificateRealm

  • Proprietà specifiche della classe JDBCRealm

  • Proprietà specifiche della classe LDAPRealm

  • Proprietà specifiche della classe SolarisRealm

  • Proprietà specifiche della classe PamRealm

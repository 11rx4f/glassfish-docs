Per visualizzare il blog di GlassFish Server

Il blog di GlassFish Server fornisce notizie sulla community di GlassFish Server e collegamenti a informazioni aggiuntive su GlassFish Server.

  1. Selezionare il nodo Task comuni nell'albero di navigazione.

    Viene visualizzata la pagina Task comuni.

  2. In Notizie su GlassFish fare clic sul collegamento Blog.

    Il blog di GlassFish Server viene visualizzato in una scheda o finestra distinta del browser.

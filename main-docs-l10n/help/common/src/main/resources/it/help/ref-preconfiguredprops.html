Proprietà di sistema predefinite

Di seguito sono riportate le proprietà di sistema predefinite per GlassFish Server.

com.sun.aas.installRoot

La directory di installazione di GlassFish Server.

com.sun.aas.instanceRoot

La directory di livello superiore per un dominio GlassFish Server.

com.sun.aas.hostName

Il nome dell'host (computer).

com.sun.aas.javaRoot

La directory di installazione di Java Runtime.

com.sun.aas.imqLib

La directory libreria per il software Oracle Message Queue.

com.sun.aas.domainName

Il nome del dominio. Questa proprietà non viene utilizzata nella configurazione predefinita, ma può essere utilizzata per personalizzare la configurazione. Il valore predefinito è domain1.

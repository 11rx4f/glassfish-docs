Proprietà specifiche dei moduli di audit

Per un modulo di audit è disponibile la proprietà riportata di seguito.

auditOn

Il valore true comporta il caricamento del modulo di audit e assicura che il modulo venga richiamato dalla libreria di audit di GlassFish Server in corrispondenza dei punti di audit. Il valore false disabilita il log di audit per il modulo.

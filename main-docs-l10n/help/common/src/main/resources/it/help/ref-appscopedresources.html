Risorse con ambito applicazione

Utilizzare la pagina Risorse con ambito applicazione per visualizzare le risorse disponibili solo per una determinata applicazione.

La pagina Risorse con ambito applicazione contiene una tabella con le informazioni riportate di seguito.

Nome applicazione

Il nome dell'applicazione.

Nome modulo

Il nome del modulo all'interno dell'applicazione.

Nome risorsa

Il nome della risorsa con ambito applicazione per l'applicazione o il modulo. Fare clic sul nome per modificare le impostazioni della risorsa.

Tipo di risorsa

Il tipo di risorsa, ad esempio Connection pool JDBC o Risorse connettore.

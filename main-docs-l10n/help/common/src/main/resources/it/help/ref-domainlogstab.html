Log di dominio

Utilizzare la pagina Log di dominio per raccogliere e scaricare i file di log per l'istanza o il cluster GlassFish Server specificato e per raggruppare i log in un unico archivio ZIP.

Nota:

Se si specifica un cluster, l'archivio ZIP includerà anche il file di log per il server di amministrazione del dominio (DAS).

La pagina Log di dominio contiene le opzioni riportate di seguito.

Cluster

L'elenco a discesa dei cluster, se esistenti.

Istanza

L'elenco a discesa delle istanze server.

Raccogli log

Pulsante che consente di raccogliere i log in un file log-files-nome.zip file, dovenome è il nome del cluster o dell'istanza.

Il file log-files-nome.zip viene scaricato utilizzando il meccanismo di download del browser.

Task correlati

  • Per raccogliere e scaricare i log del dominio

  • Per visualizzare i file di log del server DAS

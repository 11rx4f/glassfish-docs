Per configurare le opzioni della JVM

Utilizzare la pagina Opzioni JVM per specificare le opzioni del launcher dell'applicazione Java (java) che esegue GlassFish Server. Le opzioni -D indicano le proprietà specifiche di GlassFish Server.

  1. Selezionare il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM fare clic sulla scheda Opzioni JVM.

    Viene visualizzata la pagina Opzioni JVM.

  5. Per modificare un'opzione, è necessario modificare il valore del campo Valore dell'opzione.

  6. Per aggiungere un'opzione:

    1. Fare clic sul pulsante Aggiungi opzione JVM.

    2. Nella riga vuota visualizzata, digitare le informazioni sull'opzione nel campo Valore.

      Se le informazioni sull'opzione contengono uno o più spazi, racchiudere l'intera stringa tra virgolette.

  7. Per eliminare un'opzione:

    1. Selezionare la casella di controllo a sinistra del campo Valore corrispondente all'opzione da eliminare.

    2. Fare clic su Elimina.

  8. Fare clic su Salva.

  9. Riavviare il server.

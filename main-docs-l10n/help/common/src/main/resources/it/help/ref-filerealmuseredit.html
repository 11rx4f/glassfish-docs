Modifica utente realm

Utilizzare la pagina Modifica utente realm per modificare la password di un utente per il realm corrente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile modificare la password di un utente di admin-realm impostando un valore vuoto.

La pagina Modifica utente realm contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome realm

Il nome del realm attualmente selezionato. Questo campo è di sola lettura.

ID utente

Il nome dell'account da modificare. Questo campo è di sola lettura.

Lista gruppi

I gruppi ai quali appartiene l'utente corrente. Separare i nomi dei gruppi con una virgola.

Per gli utenti admin-realm, questo campo è di sola lettura e il gruppo è asadmin.

Nuova password

La nuova password per l'utente.

Conferma nuova password

La nuova password per l'utente.

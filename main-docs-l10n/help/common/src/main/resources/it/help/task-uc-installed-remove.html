Per visualizzare i componenti installati

La pagina Componenti installati fornisce informazioni sui componenti dell'immagine di installazione installati.

La disinstallazione di un componente implica la rimozione degli elementi riportati di seguito.

  • Tutti i file e le directory introdotti con l'installazione originale del componente

  • Tutti i file e le directory introdotti con gli aggiornamenti del componente dopo l'installazione originale

  1. Selezionare il nodo Strumento di aggiornamento nell'albero di navigazione.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili fare clic sulla scheda Componenti installati.

    Viene visualizzata la pagina Componenti installati.

Passi successivi

Per disinstallare i componenti installati, è necessario in primo luogo arrestare il dominio di GlassFish Server. Utilizzare quindi uno dei comandi riportati di seguito.

  • as-install-parent/bin/updatetool, che consente di avviare la utility grafica Strumento di aggiornamento.

  • as-install-parent/bin/pkg, la versione da riga di comando dello strumento di aggiornamento.

Per informazioni sull'uso del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. La utility Strumento di aggiornamento dispone di una Guida in linea specifica.

Una volta completata la disinstallazione dei componenti, riavviare il server.

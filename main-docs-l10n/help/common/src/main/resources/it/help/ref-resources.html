Risorse

Utilizzare la pagina Risorse per configurare le risorse seguenti per l'istanza o il cluster di GlassFish Server selezionato:

  • JDBC

  • Connettori

  • Configurazioni Resource Adapter

  • Risorse JMS

  • Sessioni Jakarta Mail

  • JNDI

  • Risorse concorrenti

La pagina Risorse contiene inoltre l'opzione seguente:

Aggiungi risorse

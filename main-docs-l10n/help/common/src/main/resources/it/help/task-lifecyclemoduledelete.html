Per eliminare un modulo del ciclo di vita

  1. Selezionare il nodo Moduli del ciclo di vita nell'albero di navigazione.

    Viene visualizzata la pagina Moduli del ciclo di vita.

  2. Nella pagina Moduli del ciclo di vita selezionare la casella di controllo a sinistra del nome del modulo del ciclo di vita da eliminare.

  3. Fare clic su Elimina.

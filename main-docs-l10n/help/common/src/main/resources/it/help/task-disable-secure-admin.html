Per disabilitare l'amministrazione sicura

  1. Selezionare il nodo server (server di amministrazione) nell'albero di navigazione.

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali fare clic sul pulsante Amministrazione sicura.

    Viene visualizzata la pagina Amministrazione sicura.

  3. Fare clic su Disabilita amministrazione sicura.

    Viene visualizzato un messaggio che indica che la disabilitazione della amministrazione sicura comporta il riavvio immediato del server DAS.

  4. Fare clic su OK.

    L'amministrazione sicura viene disabilitata e viene visualizzata la pagina di riavvio del server.

Configurazione

Utilizzare il nodo di configurazione per un'istanza o un cluster di GlassFish Server per configurare i componenti seguenti:

  • Impostazioni generali JVM

  • Impostazioni logger

  • Contenitore Web

  • Contenitore EJB

  • Java Message Service

  • Sicurezza

  • Servizio disponibilità

  • Servizio transazioni

  • Servizio HTTP

  • Server virtuali

  • Configurazione di rete

  • Pool di thread

  • ORB

  • Servizio di amministrazione

  • Monitoraggio

  • Group Management Service

  • Proprietà di sistema

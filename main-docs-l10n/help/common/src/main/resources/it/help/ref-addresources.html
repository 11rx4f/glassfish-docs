Aggiungi risorse

Utilizzare la pagina Aggiungi risorse per creare una o più risorse da un file glassfish-resources.xml nella destinazione specificata.

La pagina Aggiungi risorse contiene le informazioni riportate di seguito.

Posizione

La posizione del file glassfish-resources.xml:

File XML da caricare nel server

Il percorso del file su un computer remoto.

File XML locale accessibile da GlassFish Server

Il percorso del file sul computer del server di amministrazione del dominio (DAS).

Destinazione

Elenco a discesa dei cluster e delle istanze server standalone.

Task correlati

Per aggiungere le risorse configurate in un file XML

Comandi asadmin correlati

Impostazioni profiler JVM

Uno strumento profiler genera dati utilizzabili per analizzare le prestazioni e identificare potenziali punti critici.

Specificare le impostazioni del profiler. Le informazioni specificate dipendono dal prodotto profiler in uso. Per esempi e istruzioni, vedere il Capitolo 3, Debugging Applications, in Oracle GlassFish Server Application Development Guide.

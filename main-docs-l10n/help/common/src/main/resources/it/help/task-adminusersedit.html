Per modificare le informazioni utente del realm admin

  1. Nel nodo Configurazioni espandere la configurazione dell'istanza server o del cluster per cui si desidera gestire gli utenti del realm admin.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Modifica realm.

  5. Nella pagina Modifica realm fare clic sul pulsante Gestisci utenti.

    Viene visualizzata la pagina Utenti amministratori.

  6. Nella colonna ID utente della tabella della pagina Utenti amministratori, fare clic sul nome dell'utente da modificare.

    Viene visualizzata la pagina Modifica utente realm.

  7. Modificare il campo Lista gruppi per impostare i gruppi di appartenenza dell'utente.

    Utilizzare le virgole per separare i nomi dei gruppi. Non è necessario definire altrove tali gruppi.

  8. Digitare una nuova password nei campi Nuova password e Conferma nuova password per modificare la password dell'utente.

  9. Fare clic su Salva.

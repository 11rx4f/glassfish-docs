Modifica alias password

Utilizzare la pagina Modifica alias password per modificare la password associata a un alias password esistente.

La pagina Modifica alias password contiene le opzioni riportate di seguito.

Nome alias

Il nome dell'alias. Questa opzione non è modificabile.

Password

La password che l'alias memorizza in forma cifrata.

Conferma password

La password che l'alias memorizza in forma cifrata. I valori Password e Conferma password vengono confrontati per assicurarsi che la password sia stata immessa senza errori di digitazione.

Per ricaricare un'applicazione distribuita

Quando si effettua il ricaricamento, l'applicazione viene semplicemente riavviata senza essere ridistribuita e vengono mantenute le impostazioni e le altre opzioni con cui l'applicazione era stata distribuita.

  1. Selezionare il nodo Applicazioni nell'albero di navigazione.

    Viene visualizzata la pagina Applicazioni.

  2. Nella tabella Applicazioni distribuite fare clic su Ricarica nella riga corrispondente all'applicazione da ridistribuire.

    Il nome applicazione può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo mediante due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata nuovamente la pagina Applicazioni, con un messaggio che indica se il ricaricamento è riuscito o meno.

Opzioni JVM

Utilizzare la pagina Opzioni JVM per specificare le opzioni del launcher dell'applicazione Java (java) che esegue GlassFish Server. Le opzioni -D indicano le proprietà specifiche di GlassFish Server.

  • Per modificare un'opzione, è necessario modificare il valore del campo Valore dell'opzione.

  • Per aggiungere un'opzione, fare clic sul pulsante Aggiungi opzione JVM. Nella riga vuota visualizzata, digitare le informazioni sull'opzione nel campo Valore.

    Se le informazioni sull'opzione contengono uno o più spazi, racchiudere l'intera stringa tra virgolette.

  • Per eliminare un'opzione, selezionare la casella di controllo a sinistra del campo Valore dell'opzione da eliminare, quindi fare clic su Elimina.

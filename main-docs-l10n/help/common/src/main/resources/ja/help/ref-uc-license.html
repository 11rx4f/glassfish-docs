ライセンス契約

「ライセンス契約」ページには、インストールまたは更新のために選択したすべてのコンポーネントに関するライセンス契約が表示されます。コンポーネントのライセンス契約は、コンポーネントの作成者によってコンポーネントとともに提供されます。ネットワーク・リポジトリの各種コンポーネントは、それぞれのライセンスに従って公開されます。例:

「ライセンス契約」ページには、次のオプションが含まれます。

受諾

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件を受諾し、インストールまたはアップグレードを実行します。

拒否

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件を拒否し、コンポーネントを選択したページに戻ります。

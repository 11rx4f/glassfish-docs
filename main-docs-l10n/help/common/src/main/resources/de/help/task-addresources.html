So werden in einer XML-Datei konfigurierte Ressourcen hinzugefügt

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Ressourcen".

    Die Seite "Ressourcen" wird geöffnet.

  2. Klicken Sie auf die Schaltfläche "Ressourcen hinzufügen".

    Die Seite "Ressourcen hinzufügen" wird geöffnet.

  3. Wählen Sie eine der folgenden Optionen, um das Verzeichnis der Datei glassfish-resources.xml anzugeben:

    XML-Datei zum Hochladen auf den Server

    Geben Sie den Pfad zu der Datei ein, wenn sie auf einem Remote-Rechner gespeichert ist.

    Lokale XML-Datei, auf die mit GlassFish Server zugegriffen werden kann.

    Geben Sie den Pfad zu der Datei ein, wenn sie auf dem Domain Administration Server-(DAS-)Rechner gespeichert ist.

  4. Wählen Sie in der Dropdown-Liste "Ziel" das Cluster oder die Standalone-Serverinstanz, in dem bzw. der die Ressourcen erstellt werden sollen.

  5. Klicken Sie auf "OK".

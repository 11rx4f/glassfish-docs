Allgemeine Aufgaben

Die Seite "Allgemeine Aufgaben" enthält Shortcuts für allgemeine Administrationskonsole-Aufgaben.

Führen Sie über die Seite "Allgemeine Aufgaben" die folgenden Vorgänge aus:

  • So wird die Registrierungsseite für GlassFish Server angezeigt

  • So wird die Abonnementsseite für GlassFish Server angezeigt

  • So wird der GlassFish Server-Blog angezeigt

  • So wird eine Liste der bereitgestellten Anwendungen angezeigt

  • So wird eine Webanwendung bereitgestellt

  • So wird eine Unternehmensanwendung bereitgestellt

  • So wird ein Anwendungsclient bereitgestellt

  • So stellen Sie ein Connector-Modul bereit

  • So wird ein EJB-JAR bereitgestellt

  • So wird der Überwachungsservice konfiguriert

  • So werden Anwendungsüberwachungsdaten angezeigt

  • So werden Serverüberwachungsdaten angezeigt

  • So werden Ressourcenüberwachungsdaten angezeigt

  • So werden installierte Komponenten angezeigt

  • So werden Add-on-Komponenten installiert

  • So werden verfügbare Updates für Komponenten angezeigt

  • So erstellen Sie einen JDBC-Connection-Pool

Darüber hinaus finden Sie auf der Seite "Allgemeine Aufgaben" Links zur GlassFish Server-Dokumentation.

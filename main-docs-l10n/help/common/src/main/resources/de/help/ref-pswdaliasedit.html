Kennwortalias bearbeiten

Verwenden Sie die Seite "Kennwortalias bearbeiten", um das Kennwort zu ändern, das zu einem vorhandenen Kennwortalias gehört.

Die Seite "Kennwortalias bearbeiten" enthält die folgenden Optionen:

Aliasname

Der Name des Alias. Diese Option kann nicht bearbeitet werden.

Kennwort

Das Kennwort, das den Kennwortalias in verschlüsselter Form speichert.

Kennwort bestätigen

Das Kennwort, das den Kennwortalias in verschlüsselter Form speichert. Die Werte "Kennwort" und "Kennwort bestätigen" werden verglichen, um sicherzustellen, dass das Kennwort fehlerfrei eingegeben wurde.

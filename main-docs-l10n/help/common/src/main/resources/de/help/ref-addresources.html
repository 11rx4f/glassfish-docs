Ressourcen hinzufügen

Verwenden Sie die Seite "Ressourcen hinzufügen", um mindestens eine Ressource aus einer glassfish-resources.xml-Datei auf dem angegebenen Ziel zu erstellen.

Die Seite "Ressourcen hinzufügen" enthält die folgenden Informationen.

Verzeichnis

Das Verzeichnis der glassfish-resources.xml-Datei:

XML-Datei zum Hochladen auf den Server

Pfad zu der Datei auf einem Remote-Rechner.

Lokale XML-Datei, auf die mit GlassFish Server zugegriffen werden kann.

Pfad zu der Datei auf dem Rechner mit dem Domain Administration Server (DAS).

Ziel

Eine Dropdown-Liste der Cluster und Standalone-Serverinstanzen.

Verwandte Aufgaben

So werden in einer XML-Datei konfigurierte Ressourcen hinzugefügt

Zugehörige asadmin-Befehle

add-resources(1)

Neues Administratorkennwort

Auf der Seite "Neues Administratorkennwort" können Sie das Kennwort für den aktuellen Administratoraccount ändern.

Hinweis:

Wenn die sichere Administration aktiviert ist, können Sie das Kennwort nicht in einen leeren Wert ändern.

Die Seite "Neues Administratorkennwort" enthält die folgenden Optionen.

Benutzer-ID

Die Benutzer-ID des aktuellen Administrators. Dies ist ein schreibgeschütztes Feld.

Gruppenliste

Die Gruppe, zu der der Administrator gehört. Dies ist ein schreibgeschütztes Feld. Der Wert ist asadmin.

Neues Kennwort

Das neue Kennwort für den Administrator.

Neues Kennwort bestätigen

Das neue Kennwort für den Administrator.

Verbundene Aufgabe

So wird das Administratorkennwort geändert

Zugehöriger asadmin-Befehl

change-admin-password(1)

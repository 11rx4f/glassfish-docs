Providerkonfigurationen

Nutzen Sie die Seite "Providerkonfigurationen", um die Nachrichtensicherheitsprovider-Konfigurationen für die SOAP oder HttpServlet-Authentifizierungslayer zu konfigurieren.

Standardmäßig werden vier Provider für die SOAP-Authentifizierungslayer konfiguriert.

Für jede Providerkonfiguration werden die folgenden Informationen zur Verfügung gestellt.

Provider-ID

Der Name der Providerkonfiguration.

Providertyp

Folgende Providertypen sind möglich:

client

Legt fest, dass der Provider der Clientauthentifizierungsprovider ist.

server

Legt fest, dass der Provider der Serverauthentifizierungsprovider ist.

client-server

Legt fest, dass der Provider sowohl ein Client- als auch ein Serverauthentifizierungsprovider ist.

Standardprovider

Falls die Einstellung True lautet, ist der Provider der Standardprovider.

Klassenname

Der Name der Java-Implementierungsklasse des Providers.

Die Tabelle "Providerkonfigurationen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Providerkonfiguration.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Providerkonfiguration.

Lebenszyklusmodul-Ziele

Verwenden Sie die Seite "Lebenszyklusmodul-Ziele", um Zielcluster und Standalone-Serverinstanzen anzuzeigen, auf denen das Lebenszyklusmodul aktiviert werden kann.

Die Seite "Lebenszyklusmodul-Ziele" enthält die folgenden Informationen.

Zielname

Der Name des Clusters oder der Standalone-Serverinstanz.

Aktiviert

Zeigt True an, wenn das Lebenszyklusmodul auf dem Ziel aktiviert ist, und False, wenn das Lebenszyklusmodul deaktiviert ist.

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Ziele verwalten

Schaltfläche zum Verwalten von Lebenszyklusmodul-Zielen.

Weitere Aktionen

Dropdown-Liste mit folgenden Aktionen.

Aktivieren

Aktion zum Aktivieren des Lebenszyklusmoduls auf dem ausgewählten Ziel.

Deaktivieren

Aktion zum Deaktivieren des Lebenszyklusmoduls auf dem ausgewählten Ziel.

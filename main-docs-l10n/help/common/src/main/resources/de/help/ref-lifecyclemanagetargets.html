Ziele verwalten

Verwenden Sie die Seite "Ressourcenziele verwalten", um Zielcluster und Standalone-Serverinstanzen zu ändern, auf denen ein Lebenszyklusmodul aktiviert werden kann. Das Lebenszyklusmodul kann nur auf Zielen in der Spalte "Ausgewählte Ziele" aktiviert werden.

Die Seite "Ressourcenziele verwalten" enthält die folgenden Informationen.

Verfügbare Ziele

Die Cluster und Instanzen, auf denen das Lebenszyklusmodul nicht aktiviert werden kann.

Ausgewählte Ziele

Die Cluster und Instanzen, auf denen das Lebenszyklusmodul aktiviert werden kann.

Hinzufügen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Alle hinzufügen

Schaltfläche zum Verschieben aller Ziele von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Entfernen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Ausgewählte Ziele" nach "Verfügbare Ziele".

Alle entfernen

Schaltfläche zum Verschieben aller Ziele von "Ausgewählte Ziele" nach "Verfügbare Ziele".

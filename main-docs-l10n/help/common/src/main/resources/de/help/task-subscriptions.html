So wird die Abonnementsseite für GlassFish Server angezeigt

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Allgemeine Aufgaben".

    Die Seite "Allgemeine Aufgaben" wird geöffnet.

  2. Klicken Sie unter "GlassFish News" auf den Link "Abonnements".

    Die Webseite "GlassFish-Abonnements" wird in einer separaten Registerkarte oder einem separaten Fenster Ihres Browsers geöffnet.

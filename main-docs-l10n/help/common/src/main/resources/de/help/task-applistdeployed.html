So wird eine Liste der bereitgestellten Anwendungen angezeigt

  1. Wählen Sie in der Navigationsbaumstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet. In der Tabelle werden die bereitgestellten Anwendungen aufgeführt.

  2. Wählen Sie einen Modultyp aus der Dropdown-Liste "Filter", um die Anwendungsliste nach Modultyp zu filtern.

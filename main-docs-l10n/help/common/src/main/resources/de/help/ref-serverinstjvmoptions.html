JVM-Optionen

Auf der Seite "JVM-Optionen" können Sie die Optionen für das Java-Anwendungsstarttool (java) festlegen, das GlassFish Server ausführt. Über die -D-Optionen können Sie GlassFish Server-spezifische Eigenschaften festlegen.

  • Um eine Option zu ändern, bearbeiten Sie das entsprechende Feld "Wert".

  • Um eine Option hinzuzufügen, klicken Sie auf die Schaltfläche "JVM-Option hinzufügen". Geben Sie in der leeren Zeile im Feld "Wert" die Optionsinformationen ein.

    Wenn die Optionsinformationen mindestens ein Leerzeichen enthalten, setzen Sie die gesamte Zeichenfolge in doppelte Anführungszeichen.

  • Um eine Option zu löschen, aktivieren Sie das Kontrollkästchen links vom Feld "Wert" der zu löschenden Option, und klicken Sie anschließend auf "Löschen".

Verwandte Aufgaben

  • So werden allgemeine JVM-Einstellungen konfiguriert

  • So werden JVM-Pfadeinstellungen konfiguriert

  • So werden JVM-Optionen konfiguriert

  • So wird der Sicherheitsmanager deaktiviert

  • So werden JVM-Profiler-Einstellungen konfiguriert

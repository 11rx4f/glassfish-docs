Systemeigenschaften

Verwenden Sie die Seite "Systemeigenschaften", um GlassFish Server-Eigenschaften anzuzeigen und zu verwalten.

Eine Systemeigenschaft definiert einen allgemeinen Wert für eine Einstellung auf Serverebene. Sie können eine Systemeigenschaft in einem Textfeld "Admin-Konsole" referenzieren, indem Sie diese mit Dollarzeichen und geschweiften Klammern umschließen. Beispiel: Um eine Eigenschaft namens prop-name zu referenzieren, würden Sie die Syntax${prop-name} verwenden.

Für jede Eigenschaft enthält die Seite "Systemeigenschaften" die folgenden Informationen.

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft.

Beschreibung

Eine optionale Beschreibung der Eigenschaft.

Einige Systemeigenschaften werden im Voraus festgelegt, jedoch nicht auf dieser Seite angezeigt. Weitere Einzelheiten finden Sie unter Vordefinierte Systemeigenschaften.

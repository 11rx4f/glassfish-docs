Anwendungsspezifische Ressourcen

Auf der Seite "Anwendungsspezifische Ressourcen" können Sie Ressourcen anzeigen, die nur für eine bestimmte Anwendung verfügbar sind.

Die Seite "Anwendungsspezifische Ressourcen" enthält eine Tabelle mit den folgenden Informationen.

Anwendungsname

Der Name der Anwendung.

Modulname

Der Name des Moduls innerhalb der Anwendung.

Ressourcenname

Der Name der anwendungsspezifischen Ressource für die Anwendung oder das Modul. Klicken Sie auf den Namen, um die Ressourceneinstellungen zu ändern.

Ressourcentyp

Der Ressourcentyp, Beispiel: JDBC-Connection Pools oder Connector-Ressourcen.

Modifier l'alias du mot de passe

La page Modifier l'alias du mot de passe permet de modifier le mot de passe associé à un alias de mot de passe existant.

Elle contient les options suivantes :

Nom d'alias

Nom de l'alias. Cette option n'est pas modifiable.

Mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme cryptée.

Confirmer le mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme cryptée. Les valeurs Mot de passe et Confirmer le mot de passe sont comparées afin de vérifier que le mot de passe a été saisi sans erreur.

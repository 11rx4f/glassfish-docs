Affichage des mises à jour disponibles pour les composants

La page Mises à jour disponibles contient des informations sur les dernières versions des composants de l'image d'installation déjà installés.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Outil de mise à jour.

    La page Extensions disponibles apparaît.

  2. Sur la page Extensions disponibles, cliquez sur l'onglet Mises à jour disponibles.

    La page Mises à jour disponibles apparaît. Si aucune mise à jour n'est disponible, la table est vide.

Etapes suivantes

Pour installer les mises à jour disponibles, commencez par arrêter le domaine GlassFish Server. Utilisez ensuite l'une des commandes suivantes :

  • as-install-parent/bin/updatetool, qui démarre l'utilitaire graphique de l'outil de mise à jour

  • as-install-parent/bin/pkg, version de ligne de commande de l'outil de mise à jour

Pour plus d'informations sur l'utilisation de la commande pkg, reportez-vous au chapitre 11, Extending and Updating GlassFish Server, dans le manuel Oracle GlassFish Server Administration Guide. L'utilitaire de l'outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé l'installation des mises à jour.

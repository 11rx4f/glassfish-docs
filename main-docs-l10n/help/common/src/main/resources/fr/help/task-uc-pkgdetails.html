Affichage des informations détaillées sur un composant

L'Outil de mise à jour contient des informations détaillées sur les composants installés, les mises à jour disponibles pour ces composants et les composants d'extension.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Outil de mise à jour.

    La page Extensions disponibles apparaît.

  2. Cliquez sur l'onglet de la page où le composant est répertorié.

    Le composant est répertorié sur l'une des pages suivantes :

    • Extensions disponibles

    • Mises à jour disponibles

    • Composants installés

  3. Sélectionnez le composant dans la table des composants.

    La page Détails du composant du composant sélectionné apparaît.

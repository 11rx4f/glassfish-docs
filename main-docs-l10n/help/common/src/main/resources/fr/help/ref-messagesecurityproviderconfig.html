Configurations de fournisseur

La page Configurations de fournisseur permet de définir les configurations de fournisseur de sécurité des messages pour la couche d'authentification SOAP ou HttpServlet.

Par défaut, quatre fournisseurs sont configurés pour la couche d'authentification SOAP.

Pour chaque configuration de fournisseur, les informations ci-après sont fournies.

ID fournisseur

Nom de la configuration de fournisseur.

Type de fournisseur

Type du fournisseur, parmi les suivants :

client

Spécifie que le fournisseur est le fournisseur d'authentification côté client.

server

Spécifie que le fournisseur est le fournisseur d'authentification côté serveur.

client-server

Spécifie que le fournisseur est à la fois le fournisseur d'authentification côté client et côté serveur.

Fournisseur par défaut

Si la valeur est true, le fournisseur est le fournisseur par défaut.

Nom de classe

Nom de la classe d'implémentation Java du fournisseur.

La table Configurations de fournisseur contient également les options ci-après.

Nouveau

Bouton permettant de créer une configuration de fournisseur.

Supprimer

Bouton permettant de supprimer les configurations de fournisseur sélectionnées.

Suppression d'une configuration de fournisseur de sécurité des messages

Avant de commencer

Il n'est pas possible de créer ou de supprimer les fournisseurs de sécurité des messages pour la configuration par défaut server-config. Les seuls fournisseurs de sécurité des messages pris en charge pour la configuration par défaut server-config sont SOAP et HttpServlet. Comme ces deux fournisseurs de messages sont déjà créés par défaut lors de l'installation de GlassFish Server, le bouton Supprimer ne figure pas sur la page Configurations de la sécurité des messages pour la configuration par défaut server-config.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Développez le noeud Sécurité.

  3. Sélectionnez le noeud Sécurité des messages.

    La page Configurations de la sécurité des messages apparaît.

  4. Sur la page Configurations de la sécurité des messages, sélectionnez SOAP.

    La page Modifier la configuration de la sécurité des messages apparaît.

  5. Sur la page Modifier la configuration de la sécurité des messages, cliquez sur l'onglet Fournisseurs.

    La page Modifier la configuration du fournisseur apparaît.

  6. Sur la page Modifier la configuration du fournisseur, cochez la case située sur la gauche de la configuration de fournisseur à supprimer.

  7. Cliquez sur Supprimer.

Affichage d'informations générales pour le serveur DAS

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, procédez à l'une des étapes ci-après.

    • Consultez l'affichage du statut en lecture seule pour voir si le serveur est en cours d'exécution.

    • Consultez l'affichage du temps d'activité en lecture seule pour voir le nombre d'heures et de minutes pendant lesquelles le serveur a été continuellement en cours d'exécution.

    • Cliquez sur le bouton Arrêter pour arrêter GlassFish Server.

    • Cliquez sur le bouton Redémarrer pour redémarrer GlassFish Server.

    • Cliquez sur le bouton Afficher les fichiers journaux pour visualiser les fichiers journaux des clusters ou des instances GlassFish Server.

      Pour plus d'informations, reportez-vous à Affichage de fichiers journaux pour le serveur DAS.

    • Cliquez sur Rapport JVM pour ouvrir une fenêtre distincte qui permet d'afficher des rapports sur la Java Virtual Machine, notamment un rapport récapitulatif, un rapport sur la gestion de la mémoire, un rapport de chargement de classe et un vidage des threads en cours.

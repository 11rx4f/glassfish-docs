Activation de l'administration sécurisée

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, cliquez sur le bouton Administration sécurisée.

    La page Administration sécurisée apparaît.

  3. Si l'installation GlassFish Server utilise des valeurs d'alias d'instance ou d'administration personnalisées, mettez à jour les valeurs par défaut affichées dans les champs Alias d'administration et Alias d'instance si nécessaire.

  4. Cliquez sur Activer l'administration sécurisée.

    Un message s'affiche et indique que l'activation de l'administration sécurisée entraînera le redémarrage immédiat du serveur DAS.

  5. Cliquez sur OK.

    L'administration sécurisée est activée et la page de redémarrage du serveur apparaît.

Propriétés système

La page Propriétés système permet d'afficher et de gérer les propriétés de GlassFish Server.

Une propriété système définit une valeur commune pour un paramètre au niveau du serveur. Vous pouvez faire référence à une propriété système dans un champ textuel de la console d'administration en l'entourant du signe dollar et d'accolades. Par exemple, pour faire référence à une propriété appelée prop-name, vous utiliseriez la syntaxe ${prop-name}.

Pour chaque propriété, la page Propriétés système contient les informations ci-après.

Nom

Nom de la propriété.

Valeur

Valeur de la propriété.

Description

Description facultative de la propriété.

Un certain nombre de propriétés système sont prédéfinies mais n'apparaissent pas sur cette page. Pour plus d'informations, reportez-vous à Propriétés système prédéfinies.

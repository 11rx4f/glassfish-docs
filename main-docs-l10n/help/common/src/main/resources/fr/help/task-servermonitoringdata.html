Affichage des données de surveillance de serveur

Avant de commencer

Avant d'afficher les données de surveillance, vous devez configurer celle-ci. Pour plus d'informations, reportez-vous à Configuration du service de surveillance.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Données de surveillance.

    La page Surveillance apparaît.

  2. Dans la colonne Afficher les données de surveillance, cliquez sur le lien Serveur correspondant à l'instance de serveur que vous surveillez.

    La page Surveillance de serveur apparaît.

  3. Dans la liste déroulante Afficher, sélectionnez le composant pour lequel vous souhaitez afficher les données de surveillance.

  4. Cliquez sur le bouton Actualiser pour mettre à jour les données de surveillance affichées.

Procédure alternative :

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, cliquez sur l'onglet Surveiller.

    La page Surveillance d'application apparaît.

  3. Cliquez sur l'onglet Serveur.

    La page Surveillance de serveur apparaît.

  4. Dans la liste déroulante Afficher, sélectionnez le composant pour lequel vous souhaitez afficher les données de surveillance.

  5. Cliquez sur le bouton Actualiser pour mettre à jour les données de surveillance affichées.

Configuration des options JVM

La page Options JVM permet de spécifier les options du lanceur de l'application Java (java) exécutant GlassFish Server. Les options -D désignent les propriétés propres à GlassFish Server.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Configuration.

  2. Sélectionnez la configuration à modifier.

  3. Sous le noeud Configuration, sélectionnez le noeud Paramètres JVM.

    La page Paramètres généraux JVM apparaît.

  4. Dans la page Paramètres généraux JVM, cliquez sur l'onglet Options JVM.

    La page Options JVM apparaît.

  5. Pour modifier une option, modifiez le contenu du champ Valeur.

  6. Pour ajouter une option, procédez comme suit :

    1. Cliquez sur le bouton Ajouter une option JVM.

    2. Sur la ligne vide qui apparaît, saisissez les informations d'option dans le champ Valeur.

      Si les informations d'option contiennent des espaces, mettez la totalité de la chaîne entre guillemets.

  7. Pour supprimer une option, procédez comme suit :

    1. Cochez la case située à gauche du champ Valeur de l'option à supprimer.

    2. Cliquez sur Supprimer.

  8. Cliquez sur Enregistrer.

  9. Redémarrez le serveur.

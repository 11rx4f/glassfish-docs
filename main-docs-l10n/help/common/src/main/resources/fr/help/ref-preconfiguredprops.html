Propriétés système prédéfinies

Les propriétés système suivantes sont prédéfinies pour GlassFish Server.

com.sun.aas.installRoot

Répertoire dans lequel GlassFish Server est installé.

com.sun.aas.instanceRoot

Répertoire de niveau supérieur pour un domaine GlassFish Server.

com.sun.aas.hostName

Nom de l'hôte (ordinateur).

com.sun.aas.javaRoot

Répertoire d'installation de l'exécution Java.

com.sun.aas.imqLib

Répertoire de la bibliothèque pour le logiciel Oracle Message Queue.

com.sun.aas.domainName

Nom du domaine. Cette propriété n'est pas utilisée dans la configuration par défaut, mais elle peut l'être pour personnaliser la configuration. La valeur par défaut est domain1.

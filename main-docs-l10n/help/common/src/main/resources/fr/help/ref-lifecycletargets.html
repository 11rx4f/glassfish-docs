Cibles de module de cycle de vie

La page Cibles de module de cycle de vie permet d'afficher les clusters et les instances de serveur autonomes cible sur lesquels le module de cycle de vie peut être activé.

Elle contient les informations ci-après.

Nom de cible

Nom du cluster ou de l'instance de serveur autonome.

Activé

Affiche true si le module de cycle de vie est activé sur la cible, ou false si le module de cycle de vie est désactivé.

La table Cibles contient également les options ci-après.

Gérer les cibles

Bouton permettant de gérer les cibles de module de cycle de vie.

Plus d'actions

Liste déroulante des actions ci-après.

Activer

Action permettant d'activer le module de cycle de vie sur la cible sélectionnée.

Désactiver

Action permettant de désactiver le module de cycle de vie sur la cible sélectionnée.

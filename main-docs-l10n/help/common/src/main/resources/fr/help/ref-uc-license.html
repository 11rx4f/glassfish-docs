Contrat de licence

La page Contrat de licence affiche les contrats de licence de tous les composants sélectionnés en vue d'une installation ou d'une mise à jour. Le contrat de licence d'un composant est fourni avec ce dernier par le fabricant. Différents composants du Référentiel réseau peuvent être publiés sous différentes licences, par exemple :

La page Contrat de licence contient les options suivantes :

Accepter

Ce bouton permet d'accepter les conditions des licences de tous les composants sélectionnés, puis de poursuivre l'installation ou la mise à jour.

Refuser

Ce bouton permet de refuser les conditions des licences de tous les composants sélectionnés, puis de revenir à la page sur laquelle vous avez sélectionné les composants.

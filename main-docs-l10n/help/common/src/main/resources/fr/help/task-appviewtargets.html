Affichage des cibles d'application

  1. Dans l'arborescence de navigation, sélectionnez le noeud Applications.

    La page Applications apparaît.

  2. Sélectionnez l'application dont vous souhaitez afficher les cibles.

    Le nom de l'application peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à Module and Application Versions dans le manuel Oracle GlassFish Server Application Deployment Guide.

    La page de l'application apparaît.

  3. Sélectionnez l'onglet Cible.

    La page Cibles de l'application apparaît et les cibles actuellement spécifiées pour l'application s'affichent.

  4. Pour activer ou désactiver une application sur une instance ou un cluster cible spécifique, cochez la case située en regard de la cible et sélectionnez Activer ou Désactiver dans le menu déroulant Plus d'actions.

  5. Pour permettre l'équilibrage de charge d'une cible sur une instance ou un cluster cible spécifique, cochez la case située en regard de la cible et sélectionnez Equilibreur de charge activé ou Equilibreur de charge désactivé dans le menu déroulant Plus d'actions.

  6. Pour associer une application à des serveurs virtuels spécifiques, sélectionnez Gérer les serveurs virtuels pour l'instance ou le cluster cible de votre choix.

    La page Cibles de serveur virtuel apparaît.

    Pour ajouter des serveurs virtuels cible, sélectionnez-les dans la liste Cibles disponibles et déplacez-les vers la liste Cibles sélectionnées à l'aide du bouton Ajouter.

    Par défaut, le module ou l'application est associé au serveur virtuel par défaut, nommé server. Utilisez le bouton Tout ajouter pour ajouter toutes les cibles. Utilisez les boutons Enlever et Tout enlever pour enlever des cibles de la liste Cibles sélectionnées.

    Cliquez sur Enregistrer pour revenir à la page Cibles de l'application.

  7. Pour ajouter ou supprimer des instances ou clusters cible de l'application, sélectionnez Gérer les cibles.

    La page Gérer les cibles apparaît.

    Pour ajouter des cibles, sélectionnez-les dans la liste Cibles disponibles et déplacez-les vers la liste Cibles sélectionnées à l'aide du bouton Ajouter.

    Utilisez le bouton Tout ajouter pour ajouter toutes les cibles. Utilisez les boutons Enlever et Tout enlever pour enlever des cibles de la liste Cibles sélectionnées.

    Cliquez sur Enregistrer pour revenir à la page Cibles de l'application.

  8. Cliquez sur Enregistrer.

    L'application est désormais disponible dans la liste révisée des cibles.

Affichage de la page Inscription pour GlassFish Server

La page Inscription de GlassFish Server permet d'inscrire votre installation GlassFish Server.

En tant qu'utilisateur inscrit de GlassFish Server, vos avantages sont les suivants :

  • Notification sur les nouvelles versions, les patches et les mises à jour

  • Offres spéciales sur les produits, services et formation développeur

  • Accès à des livres blancs, articles pédagogiques et séminaires en ligne en direct avec l'équipe produit de GlassFish

  1. Dans l'arborescence de navigation, sélectionnez le noeud Tâches courantes.

    La page Tâches courantes apparaît.

  2. Sous Actualités de GlassFish, cliquez sur le lien Inscription.

    La page Web Inscription de GlassFish apparaît dans un nouvel onglet ou une nouvelle fenêtre du navigateur.

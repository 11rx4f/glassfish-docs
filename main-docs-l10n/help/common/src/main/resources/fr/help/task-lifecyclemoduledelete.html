Suppression d'un module de cycle de vie

  1. Dans l'arborescence de navigation, sélectionnez le noeud Modules de cycle de vie.

    La page Module de cycle de vie apparaît.

  2. Sur la page Modules de cycle de vie, cochez la case située à gauche du module de cycle de vie à supprimer.

  3. Cliquez sur Supprimer.

Ressources de niveau application

La page Ressources de niveau application permet de visualiser les ressources disponibles uniquement pour une application spécifique.

Elle contient une table comportant les informations ci-après.

Nom de l'application

Nom de l'application.

Nom du module

Nom du module dans l'application.

Nom de ressource

Nom de la ressource de niveau application pour le module ou l'application. Cliquez sur le nom pour modifier les paramètres de ressource.

Type de ressource

Type de ressource (par exemple, des pools de connexions JDBC ou des ressources de connecteur).

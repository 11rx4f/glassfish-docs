Configuration des paramètres de l'analyseur de performances JVM

Un analyseur de performances génère des données servant à analyser les performances et à identifier les éventuels goulots d'étranglement.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Configuration.

  2. Sélectionnez la configuration à modifier.

  3. Sous le noeud Configuration, sélectionnez le noeud Paramètres JVM.

    La page Paramètres généraux JVM apparaît.

  4. Sur la page Paramètres généraux JVM, cliquez sur l'onglet Analyseur de perfomarnces.

    La page Paramètres de l'analyseur de perfomarnces JVM apparaît.

  5. Définissez les paramètres de cet outil.

    Les informations saisies dépendent de l'analyseur de performances utilisé. Vous trouverez des exemples et des instructions au chapitre 3, Debugging Applications, du manuel Oracle GlassFish Server Application Development Guide.

  6. Cliquez sur Enregistrer.

  7. Redémarrez le serveur.

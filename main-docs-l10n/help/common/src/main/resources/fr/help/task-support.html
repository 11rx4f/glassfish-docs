Affichage du blog GlassFish Server

Le blog GlassFish Server contient les actualités de la communauté GlassFish Server et des liens vers des informations supplémentaires sur GlassFish Server.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Tâches courantes.

    La page Tâches courantes apparaît.

  2. Sous Actualités de GlassFish, cliquez sur le lien Blog.

    Le blog GlassFish Server apparaît dans un nouvel onglet ou une nouvelle fenêtre du navigateur.
